# Как вручную настроить интеграцию с Ozon

Интеграция с магазином на Ozon позволяет платформе «Такпродам» осуществлять продвижение ваших товаров в системе, а также:

- автоматически загружать список ваших товаров в каталог «Такпродам» и ежедневно актуализировать его;
- получать информацию о заказах ваших товаров на WB для ведения статистики и финансового учета.


Для ручной настройки интеграции нужны:

1. Seller API: Client ID и API key
2. Performance API: Client ID и Client Secret


Об автоматической настройке интеграции через расширение читайте в [статье](https://support.admitad.ru/knowledge-base/article/%D0%B0%D0%B2%D1%82%D0%BE%D0%BC%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F-%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0-%D0%B8%D0%BD%D1%82%D0%B5%D0%B3%D1%80%D0%B0%D1%86%D0%B8%D0%B8).

## Как получить Seller API: Client ID и API key

Чтобы получить Client ID:

1. В левом меню в разделе «API Интеграции» выберите [Seller API](https://seller.ozon.ru/app/settings/seller-api).
2. Скопируйте Client ID.
3. Вставьте ID в поле Client ID в форме интеграции «Такпродам».

Чтобы получить API key:

1. На странице [Seller API](https://seller.ozon.ru/app/settings/seller-api) нажмите «Сгенерировать ключ».
2. Заполните форму:  
1\) задайте название ключа;  
2\) выберите тип токена Admin read only;  
3\) нажмите «Сгенерировать».
3. Скопируйте созданный ключ доступа и вставьте его в поле API key в форме интеграции «Такпродам».


Важно! Ключ можно скопировать только сразу после его создания. Сохраните его, так как повторное копирование невозможно.

## Как получить Performance API: Client ID и Client Secret

1. В левом меню в разделе «API Интеграции» выберите [Performance API](https://seller.ozon.ru/app/settings/performance-api).
2. На странице Performance API нажмите «Создать аккаунт». Новый аккаунт появится в списке.
3. Нажмите на номер созданного аккаунта в колонке «Сервисный аккаунт».
4. На странице сервисного аккаунта нажмите «Добавить ключ».
5. Задайте название ключа и нажмите «Добавить». Ключ появится на странице сервисного аккаунта.
6. Скопируйте Client ID и Client Secret для созданного ключа.
7. Вставьте ID и Secret в соответствующие поля в форме интеграции «Такпродам».
