# Что такое промо-витрина

Витрина — это рекламная страница с подборкой товаров от определенного магазина на маркетплейсе.


В разделе «Каталог товаров» → «Промо\-витрины» товары, которые доступны для продвижения в рамках CPA и CPC кампаний, представлены в виде витрин.


Для каждой витрины указаны параметры:

- модель оплаты,
- потенциальная комиссия за клик или заказ,
- категория товаров,
- дата создания,
- ссылка.


Использование промо\-страниц в рекламе облегчает поиск и выбор товаров для пользователя, стимулирует кросс\-продажи. Продвигайте витрины на своих площадках, чтобы увеличить вероятность клика и покупки товара по ссылке.


Важно:

- Витрины расположены в порядке создания, начиная с самых новых.
- Список витрин можно отфильтровать, учитывая категорию товаров, модель оплаты и маркетплейс.
- Если хотите продвигать отдельный товар витрины, копируйте его артикул. В каталоге СРА или СРС товаров через поиск по артикулу найдите нужный товар и партнерскую ссылку для размещения.
- При публикации креатива на своей площадке укажите данные для маркировки рекламы из карточки витрины.
