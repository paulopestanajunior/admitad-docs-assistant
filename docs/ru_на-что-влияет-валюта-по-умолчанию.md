# На что влияет валюта по умолчанию

В разделе «Общие настройки аккаунта» вашего аккаунта есть параметр «Валюта по умолчанию».  
Значение этого параметра определяется при регистрации и зависит от вашей страны и языка.

## На что влияет валюта по умолчанию?


Валюта по умолчанию влияет только на то, в какой валюте будет рассчитываться и отображаться статистика в двух разделах вашего аккаунта:

- В блоке «Статистика за 30 дней» на главной странице. В валюте по умолчанию отображается ваш заработок.
- В разделе «Статистика» во всех типах статистики, кроме статистики по действиям.


Здесь валюта по умолчанию используется для расчета и отображения eCPC и eCPM и заработка в колонке «Финансы»:

## На что не влияет валюта по умолчанию?

- На отображение баланса в блоке «Ваш аккаунт».  
Баланс отображается в тех валютах, в которых вы получаете вознаграждение по партнерским программам, с которыми работаете.
- На выплату заработка.  
При заказе выплаты ваш заработок не будет автоматически конвертирован в валюту по умолчанию. В процессе вывода средств вам будут предложены доступные варианты вывода, в том числе с конвертацией в нужную вам валюту. О конвертации валют читайте [здесь](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%BE%D0%BD%D0%B2%D0%B5%D1%80%D1%82%D0%B0%D1%86%D0%B8%D1%8F-%D0%B2%D0%B0%D0%BB%D1%8E%D1%82%D1%8B-%D0%BF%D1%80%D0%B8-%D0%B2%D1%8B%D0%B2%D0%BE%D0%B4%D0%B5-%D0%B4%D0%B5%D0%BD%D0%B5%D0%B3_1).
- На статистику по действиям.  
Валюта по умолчанию не используется в статистике по действиям.  
Здесь сумма заказа и заработок отображаются в валюте партнерской программы, по которой было совершено действие:

## Как изменить валюту по умолчанию?


1\. Наведите курсор на свой логин в правом верхнем углу и перейдите в настройки аккаунта.


2\. В разделе «Общие настройки аккаунта» найдите поле «Валюта по умолчанию» и нажмите «Редактировать».

  
3\. Выберите нужную валюту из выпадающего списка и нажмите «Сохранить».

  
После сохранения валюта в разделах статистики изменится на выбранную вами.
