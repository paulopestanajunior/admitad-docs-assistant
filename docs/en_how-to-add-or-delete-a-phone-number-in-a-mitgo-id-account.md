# How to add or delete a phone number in a Mitgo ID account

A phone number may help verify it's you who owns the account.

If you lose access to the email linked to your account, a phone number can make it faster for you to restore access to your account.  
  

In your Mitgo ID account, you can:

- [Add a phone number](#adding-phone-number)
- [Delete the current phone number](#deleting-phone-number).

To change the phone number, remove the current number first, then add a new one.

## How to add a phone number to your Mitgo ID account

1\. In the [Security](https://id.mitgo.com/security) section, click the Phone number field in Email and phone number.

2\. You'll receive an email with a code to confirm the request to add a phone number.  
[What to do if you didn't get the email](https://support.mitgo.com/hc/en-us/articles/12323285787025#did-not-receive-confirmation-email)

3\. On the page that opens in the Mitgo ID account, enter the code from the email you received and click Confirm.

4\. On the page that opens, enter the phone number that you'd like to add and click Next.

5\. A confirmation code will be texted to the phone number you provided.

Enter the code from the text message and click Confirm.

Done! You've added the phone number to your Mitgo ID account.

If you didn't receive a text message with a code, click Start over.

## How to delete a phone number from your Mitgo ID account

1\. In the [Security](https://id.mitgo.com/security) section, click the Phone number field in Email and phone number.

2\. Click Delete phone number on the page that opens.

You'll receive an email with a code to confirm the request to delete the phone number.  
[What to do if you didn't get the email](https://support.mitgo.com/hc/en-us/articles/12323285787025#did-not-receive-confirmation-email)

3\. On the page that opens in the Mitgo ID account, enter the code from the email you received and click Confirm.

Done! Your phone number has been successfully deleted from your Mitgo ID account.

*To add a new phone number, click Add phone number and follow the [instructions above](#adding-phone-number).*
