# What amounts can be withdrawn

- The minimum amount you can withdraw from Wallet depends on the selected currency. If you try to withdraw less than the minimum, you'll see a hint reminding you of the minimum amount you can withdraw.
- The maximum amount depends on the withdrawal method. If you specify an amount that is greater than the maximum, you'll receive a message stating that you've exceeded the max.


The table below presents the minimum amounts for the most popular currencies:

| --- | --- | --- |
| US Dollar | USD | 20 USD |
| Euro | EUR | 20 EUR |
| British Pound | GBP | 50 GBP |
| Chinese Yuan | CNY | 35 CNY |
| Turkish Lira | TRY | 800 TRY |
| Indian Rupee | INR | 1,000 INR |
| Ukraine Hryvnia | UAH | 400 UAH |
| Polish Złoty | PLN | 100 PLN |
| Israeli Shekel | ILS | 25 ILS |
| Kazakhstani Tenge | KZT | 5,000 KZT |
| UAE Dirham | AED | 20 AED |
| Brazilian Real | BRL | 100 BRL |
| Mexican Peso | MXN | 400 MXN |
