# How to create a source

After you sign up for Mobmio, you can add a source. 

A source is an analytics partner integrated with your tracker that allows you to track installs and other app events.

To create a source:

1\. Go to My sources and click Create source in the upper right corner.

If you don’t have any sources added yet, you can also go to the Catalog and click Add source in the upper right corner.

2\. In the Main info about your source section, fill out the following fields:

- Tracker. Select the tracker that you use to collect and analyze data.
- Source. Enter the source from your tracker. For AppsFlyer, it is the media source PID. Example: networkx\_int.

All sources are checked for safety. You can only add those that have passed the security check. Otherwise, you will be warned the check failed and won’t be able to proceed.

- Comment. Provide additional information if needed. For example, source details, resources you plan to use, a more attractive rate or indicative media plan.

3\. In the Details about your source section, fill out the following fields:

- Verticals. Select target categories of products or services for your source.
- Geo. Select target countries.
- Types of traffic. Select traffic types suitable for your source.

4\. Click Create source. After that it will appear on the list My sources.
