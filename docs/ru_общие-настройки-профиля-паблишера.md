# Общие настройки профиля паблишера

В меню профиля нажмите «Профиль» — откроется страница с общими настройками. Здесь вы можете:

- изменить ваш email,
- добавить или изменить контактную информацию,
- сгенерировать API токен,
- удалить аккаунт.

## Как изменить email


Электронная почта, которую вы указываете при регистрации на Такпродам, автоматически сохраняется в вашем профиле. Если нужно изменить почту, то в блоке «Mitgo ID» напротив email нажмите «Редактировать» — вы будете перенаправлены на страницу вашего аккаунта Mitgo ID. [Подробнее](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B8%D0%B7%D0%BC%D0%B5%D0%BD%D0%B8%D1%82%D1%8C-email-%D0%B2-%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B5-mitgo-id_1)

## Как добавить контактную информацию


Для удобства связи обязательно добавьте хотя бы один ваш контакт в профиль:

- ссылку на Telegram или никнейм с символом @ (например, @username),
- WhatsApp,
- номер телефона,
- еще один адрес электронной почты.


Добавленную контактную информацию можно будет удалить или изменить.

## Как сгенерировать API токен


Для автоматизации процессов, интеграции с Такпродам, доступа к документации API используйте API токен.


1\. В блоке «Генерация API токена» нажмите «Сгенерировать» — откроется окно с новым токеном.


2\. Скопируйте токен и сразу сохраните его. Когда вы закроете окно, повторно скопировать токен не получится: в интерфейсе он будет отображаться в зашифрованном виде.

## Как удалить профиль


При необходимости аккаунт можно удалить. Для этого:


1\. В блоке «Действия с аккаунтом» нажмите «Удалить».


2\. В открывшемся диалоговом окне укажите причину, по которой хотите удалить аккаунт.


Готово, вы отправили запрос на удаление аккаунта.


Когда менеджер проверит информацию, он удалит ваш профиль и отправит вам уведомление об этом. Вы не сможете восстановить свой профиль, войти в него, получить информацию по нему.
