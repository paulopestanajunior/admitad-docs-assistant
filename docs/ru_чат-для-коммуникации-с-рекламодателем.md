# Чат для коммуникации с рекламодателем

Вы можете напрямую общаться с рекламодателями, которые сами инициировали чат с вами, чтобы обсудить сотрудничество, задать интересующие вас вопросы и получить на них ответ в короткие сроки. Вам больше не придется ждать ответа на электронную почту или общаться через посредника.


Если у вас есть [гостевые аккаунты](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%BE%D1%81%D1%82%D0%B5%D0%B2%D0%BE%D0%B9-%D0%B4%D0%BE%D1%81%D1%82%D1%83%D0%BF-%D0%B2-%D0%BB%D0%B8%D1%87%D0%BD%D1%8B%D0%B9-%D0%BA%D0%B0%D0%B1%D0%B8%D0%BD%D0%B5%D1%82-%D0%B2%D0%B5%D0%B1-%D0%BC%D0%B0%D1%81%D1%82%D0%B5%D1%80%D0%B0-admitad_3), у них также будет доступ к функционалу чата.

## Как открыть окно чата


Вы можете перейти в окно чата, нажав на иконку  в правом верхнем углу в шапке личного кабинета.


Вы автоматически будете перенаправлены на страницу с вашими чатами с рекламодателями.

## Поиск по чату


Если у вас много активных чатов с рекламодателями, вы можете быстро находить нужные с помощью поисковой строки.


Для этого:


1\) В поиске начните вводить название партнерской программы.


2\) Нажмите на нужное название программы в блоке слева — откроется чат с историей переписки.
