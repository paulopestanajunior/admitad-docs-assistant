# Для чего продавцу нужно указывать юридические данные?

Заполнение юридических данных обязательно в личном профиле Такпродам. Пока юридические данные не добавлены, вы не сможете запустить рекламную кампанию и начать продвижение товаров.


Юридические данные нужны:

- для заключения [оферты](https://storage.yandexcloud.net/uploads-prod/public/seller_offer.pdf);
- для автоматической маркировки рекламы;
- для выставления акта об оказании услуг Такпродам;
- для налоговой и финансовой отчетности.


О том, как заполнить юр. данные в личном профиле, читайте в [статье](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%BD%D0%B0%D1%87%D0%B0%D1%82%D1%8C-%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D1%83-%D1%81-%D1%82%D0%B0%D0%BA%D0%BF%D1%80%D0%BE%D0%B4%D0%B0%D0%BC).


Если вы индивидуальный предприниматель или самозанятый, то при заполнении данных указывайте ФИО полностью.


Ознакомиться с политикой обработки персональных данных можно на [сайте](https://takprodam.ru/privacy-policy) Такпродам.
