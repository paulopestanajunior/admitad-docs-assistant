# Настройка отображения информации в разделе «Статистика»

В разделе «Статистика» представлена статистическая информация по всем товарам, которые находятся у вас на продвижении, а именно: информация о кликах и продажах по вашим ссылкам, комиссии за продвижение и т.д.


Вы можете настроить отображение колонок в интерфейсе, чтобы легко анализировать эффективность вашей рекламы:

- скрыть колонки, которые не используете для анализа;
- настроить порядок колонок.


При экспорте статистики файл содержит настроенные вами колонки.
  
По умолчанию выбраны все колонки для отображения.

# Как скрыть колонки


1\. Откройте раздел «Статистика».


2\. Перейдите в нужный подраздел:

- CPA\-комиссия
- CPC с учетом продаж
- CPC с витриной
- Клики


3\. Нажмите «Настроить колонки».


4\. В открывшемся списке колонок снимите галочки с тех, которые не нужно отображать.


5\. Нажмите «Применить».

# Как настроить порядок колонок


1\. Откройте раздел «Статистика».


2\. Перейдите в нужный подраздел:

- CPA\-комиссия
- CPC с учетом продаж
- CPC с витриной
- Клики


3\. Нажмите «Настроить колонки» — откроется список колонок.


4\. Перетащите строки с названием колонок вверх/вниз для изменения порядка.


5\. Нажмите «Применить».
