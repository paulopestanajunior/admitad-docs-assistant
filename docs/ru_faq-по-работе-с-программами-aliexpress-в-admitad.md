# FAQ по работе с программами AliExpress в Admitad

## Когда будет обработано целевое действие по программе AliExpress?


Целевое действие будет обработано, когда покупатель в своем кабинете AliExpress нажмет «Подтвердить получение товара».


В течение нескольких часов после этого статус целевого действия по программе AliExpress автоматически изменится на «Подтвержден».


В случае, если покупатель не подтвердит получение заказа в личном кабинете AliExpress, обработка такого действия может занять продолжительное время.


Либо действие может быть отклонено по причинам, которые от не зависят от Admitad.

## Какую программу AliExpress выбрать для подключения?


В каталоге [Admitad Store](https://store.admitad.com/ru/catalog/) представлено две программы AliEхpress: AliExpress RU\&CIS NEW и AliExpress WW.


Главное отличие между ними заключается в условиях начисления вознаграждения:

- в программе AliExpress RU\&CIS NEW по полной ставке оплачиваются только целевые действия, которые совершают пользователи из следующих стран: Россия, Армения, Азербайджан, Беларусь, Грузия, Казахстан, Киргизия, Молдова, Таджикистан, Туркменистан, Узбекистан;
- в программе AliExpress WW, TR и IN оплачиваются только целевые действия, которые совершают пользователи из всех остальных стран (Европы, Америки, Азии, Африки).


Подробное описание различий программ AliExpress RU\&CIS NEW и AliExpress WW см. в [этой статье](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA%D1%83%D1%8E-%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D1%83-%D0%B2%D1%8B%D0%B1%D1%80%D0%B0%D1%82%D1%8C-aliexpress-ww-%D0%B8%D0%BB%D0%B8-aliexpress-rucis-new_1).

## В какой валюте AliExpress выплачивает вознаграждение?


Валюта зависит от выбранной вами [программы](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA%D1%83%D1%8E-%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D1%83-%D0%B2%D1%8B%D0%B1%D1%80%D0%B0%D1%82%D1%8C-aliexpress-ww-%D0%B8%D0%BB%D0%B8-aliexpress-rucis-new_1):

- Вознаграждение по программе AliExpress WW, TR и IN выплачивается в USD (долларах США).
- Вознаграждение по программе AliExpress RU\&CIS NEW выплачивается в RUB (российских рублях).

## Учитываются ли действия при заказе с мобильных устройств?


В системе учитываются все действия вне зависимости от того, с какого типа устройства они были выполнены. Учитываются действия, совершенные и в мобильном браузере, и в мобильном приложении. Действия с мобильных устройств оплачиваются по тем же тарифам, что и действия с ПК.


Если пользователь с ПК перешел по вашей партнерской ссылке на сайт AliExpress, добавил в корзину товар, а потом зашел в мобильное приложение и оплатил его, то заказ будет зафиксирован в системе, и в случае подтверждения действия вы получите вознаграждение.

## Как часто обновляется статистика по AliExpress?


Статистика по AliExpress обновляется каждые 5 минут.

## Могу ли я получить информацию по корзине?


Да, вы можете получить информацию по корзине.


В [статистике по действиям](https://store.admitad.com/ru/webmaster/statistics/actions/) («Статистика» → «Статистика по действиям») помимо стандартных столбцов передается:

- в личном кабинете — название товара и ссылка на товар.
- в XLS\-выгрузке — название товара, ссылка на изображение товара, ссылка на страницу товара, код страны товара, ID категории товара, название категории товара.
- в XML\-выгрузке — ссылка на изображение товара, ссылка на страницу товара, код страны товара, ID категории товара, название товара, название категории товара.
- в API — название товара, ID товара, ссылка на товар, ссылка на изображение товара.

  
Вы можете получать детальную статистику и работать с сегментами товаров и пользователей, направляя трафик на приоритетные или продаваемые товары.

## AliExpress поддерживает работу с потерянными заказами?


Нет, к сожалению, AliExpress не поддерживает работу с потерянными заказами.

## Начисляет ли AliExpress бонусы за новых покупателей?


С 31 мая 2022 года AliExpress не начисляет бонусы за новых покупателей.


Ранее в партнерской программе AliExpress WW выплачивалось повышенное вознаграждение за новых покупателей для кешбэк\-сервисов и браузерных расширений.

## Предоставляет ли AliExpress лицензии или гарантийные письма для рекламных сетей?


Нет, не предоставляет.
