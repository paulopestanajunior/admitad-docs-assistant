# Security settings

Enable two\-factor authentication to confirm billing details in your account. This measure will help you protect your earnings.


Two\-factor authentication is an extra level of account protection that guarantees that only you can be granted access to your account, even if someone else has your password.

## How to set up two\-factor authentication


If you have already set up two\-factor authentication [in your Mitgo ID account](https://id.mitgo.com/security/totp), you don't have to enable it again in Mitgo Wallet.

1. In [Billing details](https://earnings.admitad.com/billing-details?_gl=1*1n74zam*_ga*OTk0MTc3NjIxLjE2ODk3NjY2MTE.*_ga_32HHLCD33P*MTcwNjg2NjUzNS44LjEuMTcwNjg2Njg1NC41MC4wLjA.), click Enable in the Enable two\-factor authentication area.
2. In the window that opens, click Send verification code. The code will be sent to your email. You can request another code in one minute.
3. Enter the code and click Verify.
4. The setup page will open. Follow the prompts:


4\.1 Download the Google Authenticator app to your device:

- [Link for iOS](https://apps.apple.com/en/app/google-authenticator/id388497605)
- [Link for Android](https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2)


4\.2 Scan the QR code on the screen. Your account will be added to the Google Authenticator app, and you'll see a 6\-digit code that refreshes every 30 seconds.


4\.3 Save the key. Without it, you won't be able to reset two\-factor authentication or recover access to your account if you lose access to the authentication device.


4\.4 Enter this code in the Code field.


4\.5 Enter the name of the device on which the app is installed in the Device name field.


5\. Click Enable protection.


Done! You have set up two\-factor authentication. From now on, to confirm new billing details, you will need to enter the one\-time code from the Google Authenticator app.
