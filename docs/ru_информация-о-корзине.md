# Информация о корзине

Если на странице программы в блоке «Инструменты» вы видите инструмент «Информация о корзине», это означает, что вы можете получать данные о составе корзин (заказов) пользователей по этой программе.

  
Получить информацию о корзине можно через [этот метод API](https://developers.admitad.com/hc/ru/articles/7930541032849-%D0%A1%D1%82%D0%B0%D1%82%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0-%D0%B2%D0%B5%D0%B1-%D0%BC%D0%B0%D1%81%D1%82%D0%B5%D1%80%D0%B0#reports-by-actions), в личном кабинете она отсутствует.

Инструмент включается автоматически, если в программе выполнены два условия одновременно:

- более 50% заказов за последние 3 дня имеют product\_id;
- в программе есть хотя бы один товарный фид.

Чтобы найти программы, у которых инструмент включен, в [Admitad Store](https://store.admitad.com/ru/catalog/) отметьте галочкой опцию «Информация о корзине»:
