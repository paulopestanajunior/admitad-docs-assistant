# Cart tracking

If you see the Cart tracking tool in the Tools block on a program's page, it means that you can receive data on cart items within this program.

  
  

Information on the cart can be obtained through [this API method](https://developers.admitad.com/hc/en-us/articles/7930541032849-Publisher-reports#reports-by-actions).

The tool is automatically turned on if the following conditions are met simultaneously in the program:

- more than 50% of orders within the last 3 days have product\_id;
- the program has at least one XML feed.

To find programs that support this tool, set the Cart tracking filter in [Admitad Store](https://store.admitad.com/en/catalog/).
