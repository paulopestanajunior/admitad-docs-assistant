# How to verify an ad space

Verification is needed to confirm your ownership of the ad space and to prevent fraud. If an ad space is not verified, you cannot connect it to [Mitgo products](https://id.mitgo.com/services) for monetization.


You can complete verification when creating the ad space or later.


If you did not verify the ad space when creating it, follow these steps:


1\. Go to the [Ad spaces](https://id.mitgo.com/adspaces) section — a list of all your ad spaces in Mitgo ID will open.


2\. Next to the ad space you want to verify, click Verify.


3\. Select a verification method:

- [via meta tag](#verification-via-meta-tag),
- [via DNS record](#verification-via-DNS-record),
- [via verification code](#verification-via-code).


The available methods depend on the type of ad space.

## Verification via meta tag

This method is suitable for websites.


1\. On the Verification via meta tag page, copy the specified meta tag.


2\. Place it between the \<head\> and \</head\> tags of your website's main page.


3\. Return to the Verification via meta tag page and check the box to confirm you have completed all actions to prove ownership of the ad space.


4\. Click Finish.


If everything is done correctly, the ad space will be verified automatically within 15 minutes. It will receive the status "Active", and you will be able to connect it to Mitgo products.

## Verification via DNS record

This method is suitable for websites.


1\. On the Verification via DNS record page, copy the TXT\-type DNS record.


2\. Log in to your account on the domain registrar's website.


3\. Open the DNS records page. It may be named any of the following:

- DNS Management
- DNS
- Nameserver Management
- Control Panel
- Advanced settings


4\. Add a DNS record:

- In the Record type field, select TXT.
- In the Value/Response/Destination field, paste the TXT\-type DNS record copied in Mitgo ID.


Leave default values in the remaining fields.


5\. Save the changes.


6\. Return to the Verification via DNS record page and check the box to confirm you have completed all actions to prove ownership of the ad space.


7\. Click Finish.


Synchronization of data about the new record can take from a few minutes up to 72 hours. If everything is done correctly, the ad space will be verified automatically — it will receive the status "Active", and you will be able to connect it to Mitgo products.

## Verification via code

This method is suitable for apps and extensions.


1\. On the Verification via code page, copy the specified code.


2\. Place the code in the description of the app/extension or in the About us/About app/Information section.


Do not delete or edit the code until the ad space is approved.


3\. Return to the Verification via code page and click the Send request button.


A moderator will check for the presence and correctness of the code. The check takes from 2 up to 72 hours. After successful verification, the ad space will receive the status "Active", and you will be able to connect it to Mitgo products.

## FAQ

### Where can I see the status of my ad space?


You can see the status of your ad space on the [Ad spaces](https://id.mitgo.com/adspaces) page.

### What should I do if the ad space did not pass verification via code?


1\. Check the reason for rejection in the email sent to your registration email address.


2\. Register the ad space again and submit it for verification, taking into account the advice in the email.


If you still have questions, contact the support team — support@admitad.com

### What should I do if the ad space did not pass verification via meta tag or DNS record?


If the automatic check does not find the correct DNS record or the required meta tag on your site, the ad space will change its status to "Not verified". After this you can correct the meta tag or DNS record and start the verification again.
