# Доступы к Admitad API для веб-мастера

Всю детальную информацию по API и его использованию можно получить из [Документации Admitad API](https://developers.admitad.com/hc/ru/).


Учетные данные, которые потребуются при работе с API, находятся на вкладке «API и приложения» в настройках аккаунта.


По умолчанию учетные данные скрыты в целях безопасности. Чтобы увидеть их, нажмите на кнопку «Показать учетные данные».


Учетные данные API дают полный доступ к вашему аккаунту, так же как и ваш логин и пароль, поэтому никому их не передавайте.


Если вы знаете или подозреваете, что доступ был украден, сбросьте данные. После этого будет сгенерирован новый client\_secret. Учтите, что при сбросе все приложения, которые используют этот ключ, будут отключены.


Помимо учетных данных здесь представлен список приложений, которым вы разрешили доступ к своему аккаунту. Нажав на кнопку «Доступы» можно проверить, какие именно права доступа получило то или иное приложение.
