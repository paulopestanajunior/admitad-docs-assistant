# Статистика веб-мастера в Admitad

В статистике Admitad фиксируются все клики по баннерам и партнерским ссылкам, а также целевые действия пользователей, такие как лиды и продажи. Благодаря этим данным вы сможете анализировать эффективность вашей рекламы и отслеживать статус действий.


В зависимости от ваших целей вы можете выбрать одну из вкладок, соответствующих следующим видам статистики:

- [статистика по площадкам](#Reports-on-ad-spaces);
- [статистика по партнерским программам](#Reports-on-programs);
- [статистика по действиям](#Reports-on-actions);
- [статистика по времени](#Reports-on-time);
- [статистика по контекстным объявлениям](#Reports-on-context);
- [статистика по SubID](#Reports-on-SubID);
- [статистика по персональным промокодам](#Reports-on-take&go-promocodes).


Вы также можете посмотреть статистику за последние 7 дней по площадке или отдельным программам с любого устройства, на котором установлен Telegram, с помощью Admitad Bot. [Подробнее](https://support.admitad.ru/knowledge-base/article/admitad-bot-%D0%B4%D0%BB%D1%8F-telegram_1#get-statistics)


В верхней части каждой вкладки можно выбрать период отображения и настроить необходимые фильтры.


Статистика обновляется каждые 15\-30 минут. В случае большой нагрузки на систему обновление статистики может длиться на несколько минут дольше.

## Статистика по площадкам


В статистике по площадкам данные разделены по площадкам, что позволяет оценивать их сравнительную эффективность.


Таблица содержит следующие столбцы:

- Площадка — название площадки. При клике на название появится список всех программ, по которым за выбранный период зафиксированы клики или действия.
	- Если вы кликнете на название программы, то увидите статистику по каждому рекламному материалу этой программы (баннеру, купону или просто партнерской ссылке), который размещен на данной площадке.
	- Если кликнуть на кнопку  («Перейти к программе»), вы попадете в карточку программы.
- Просмотры — если вы разместили баннер, эта колонка покажет количество уникальных просмотров для него.
- Клики — количество кликов по рекламному материалу или ссылке.
- [CTR](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-admitad-partner-network_1#ctr-en) — процентное отношение числа кликов к числу просмотров баннера за выбранный период («Клики»/«Просмотры»×100%).
- [eCPC](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-admitad-partner-network_1#ecpc-en) — ваш заработок с 1 клика. Рассчитывается по формуле «Финансы» («Подтвержденные»)/«Клики».
- [eCPM](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-admitad-partner-network_1#ecpm-en) — ваш заработок с 1000 просмотров. Актуально только для баннеров. Рассчитывается по формуле «Финансы» («Подтвержденные»)/«Просмотры»×1000\.
- Конверсия ([CR](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-admitad-partner-network_1#cr-en)) — процентное отношение количества целевых действий пользователей к общему числу кликов. Рассчитывается по формуле («Лиды\+«Продажи»)/«Клики»×100%.
- Действия — количество целевых действий, совершенных пользователями за выбранный период. Делятся на «[Лиды](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-admitad-partner-network_1#lead-en)» и «[Продажи](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-admitad-partner-network_1#sale-en)». Кликнув по ним, вы увидите список данных действий в «Статистике по действиям».
- Финансы — ваш заработок за целевые действия, выполненные пользователями.
	- Отклоненные — вознаграждение за действия, которые были отклонены рекламодателями при обработке (см. «[Почему рекламодатель отклонил действия?](https://support.admitad.ru/knowledge-base/article/%D0%BF%D0%BE%D1%87%D0%B5%D0%BC%D1%83-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C-%D0%BE%D1%82%D0%BA%D0%BB%D0%BE%D0%BD%D0%B8%D0%BB-%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D1%8F_1)»).
	- Открытые — вознаграждение за действия, которые еще не обработаны рекламодателем.
	- Подтвержденные — вознаграждение за действия, подтвержденные рекламодателем.

## Статистика по программам


В этой вкладке данные разбиты по программам, так что вы можете сравнить эффективность сотрудничества с разными программами.

## Статистика по действиям


В «Статистике по действиям» представлен список всех зафиксированных действий по всем программам и площадкам. Здесь можно получить детальную информацию о каждом действии и, главное, отследить его статус.


Таблица содержит следующие колонки (прокрутите вправо, чтобы увидеть все):

- Программа — название партнёрской программы, по которой совершено действие.
- ID — ID действия, присваиваемый системой Admitad Partner Network.
- Площадка — ваша площадка, с которой перешел пользователь.
- Время клика — момент времени, в который произошёл переход на сайт рекламодателя.
- Время действия — момент времени, когда было совершено целевое действие на сайте рекламодателя.
- Время конверсии — период времени, прошедший с момента клика до момента действия.
- Тариф — тариф партнерской программы, к которому отнесено действие. По нему же происходит расчет вашего вознаграждения.
- Сумма заказа — сумма покупки пользователя. Не указывается для лидов.
- Статус — статус обработки действия рекламодателем. После совершения действия оно попадает в статистику со статусом «Ожидает», а вознаграждение за него появляется на главной странице в разделе «Ваш баланс» в графе «В обработке».


После подтверждения рекламодателем действие получает статус «Подтвержден», и через 1\-2 дня вознаграждение за него переходит в графу «Готовы к снятию».  
Статус «Отклонен» появляется при отклонении действия рекламодателем. Возможные причины отклонения описаны [здесь](https://support.admitad.ru/knowledge-base/article/%D0%BF%D0%BE%D1%87%D0%B5%D0%BC%D1%83-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C-%D0%BE%D1%82%D0%BA%D0%BB%D0%BE%D0%BD%D0%B8%D0%BB-%D0%B4%D0%B5%D0%B9%D1%81%D1%82%D0%B2%D0%B8%D1%8F_1).
- Заработок — ожидаемое вознаграждение, которое будет начислено при подтверждении действия. Рассчитывается по соответствующему тарифу программы.
- SubID — если к партнерской ссылке, по которой перешел покупатель, был прикреплен [SubID](https://support.admitad.ru/knowledge-base/article/subid-%D0%B2-%D0%BF%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D1%85-%D1%81%D1%81%D1%8B%D0%BB%D0%BA%D0%B0%D1%85-%D1%87%D1%82%D0%BE-%D1%8D%D1%82%D0%BE-%D0%B8-%D0%BA%D0%B0%D0%BA-%D0%B5%D0%B3%D0%BE-%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D1%8C_1), вы увидите его в этой колонке.
- Дата обработки — день, когда рекламодатель обработал действие, т.е. изменил его статус на «Подтвержден» или «Отклонен». Если статус был изменен несколько раз (например, рекламодатель по ошибке отклонил заказ, а потом подтвердил его), то указывается время последнего изменения статуса. Дата обработки не указывается для действий в статусе «Ожидает».
- Персональный промокод — если целевое действие было совершено с применением персонального промокода, здесь отобразится кодовое слово использованного промокода.
- ID заказа — номер заказа в системе рекламодателя.
- Ключевое слово — с каких ключевых слов пользователь перешел на сайт рекламодателя. Указывается только для контекстной рекламы. Подробнее см. [Статистику по контексту](#Reports-on-context)».
- Комментарий — здесь рекламодатель может оставить свой комментарий к действию (например, в случае отклонения действия или изменения суммы заказа при частичном выкупе).

## Статистика по времени


В данной вкладке можно выбрать разбивку данных по дням или месяцам.


В первом случае вы увидите статистику по каждому дню. Показываются только те дни, по которым есть какие\-либо данные.


Второй вариант удобен для выявления общих трендов, т.к. дневные колебания сглаживаются.

  
 

## Статистика по контексту


Актуальна, если вы работаете с контекстной рекламой.


На этой вкладке вы можете отследить информацию по вашим объявлениям, размещенным в различных системах контекстной рекламы. Колонки таблицы схожи с колонками «[Статистики по площадкам](https://support.admitad.ru/knowledge-base/article/%D1%81%D1%82%D0%B0%D1%82%D0%B8%D1%81%D1%82%D0%B8%D0%BA%D0%B0-%D0%B2%D0%B5%D0%B1-%D0%BC%D0%B0%D1%81%D1%82%D0%B5%D1%80%D0%B0-%D0%B2-admitad_1#Reports-on-ad-spaces)».


При клике на источник (название системы) развернется список партнерских программ, по которым вы разместили объявления в этой системе.

  
Чтобы посмотреть, с каких поисковых запросов пользователи переходили на сайт программы, нажмите на ее название. К примеру, с таких запросов пользователи переходили на сайт программы Slamdunk.


Некоторое время назад Yandex.Direct и Google Реклама изменили свою политику передачи ключевых слов, поэтому сейчас статистика может не отражать реальную ситуацию.

## Статистика по SubID


Актуальна, если вы используете в партнерских ссылках SubID для анализа трафика. На этой вкладке собраны данные по всем присвоенным SubID (даже если по ним не было кликов).

  
Пример использования статистики по SubID можно посмотреть [здесь](https://support.admitad.ru/knowledge-base/article/subid-%D0%B2-%D0%BF%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D1%85-%D1%81%D1%81%D1%8B%D0%BB%D0%BA%D0%B0%D1%85-%D1%87%D1%82%D0%BE-%D1%8D%D1%82%D0%BE-%D0%B8-%D0%BA%D0%B0%D0%BA-%D0%B5%D0%B3%D0%BE-%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D1%8C_1#examples-of-using-subid). 

## Cтатистика по персональным промокодам


На этой вкладке отображается статистика по персональным промокодам, которые вы размещаете на своей площадке, — количество целевых действий, совершенных с использованием этих промокодов, а также размер заработка по ним.


Персональные промокоды — это особый тип промокодов, которые можно размещать без партнерской ссылки. Целевые действия, которые пользователи совершают с применением таких промокодов, отслеживаются по уникальному кодовому слову. Такие промокоды рекламодатели могут предоставить по вашему запросу — [как запросить промокоды](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B7%D0%B0%D0%BF%D1%80%D0%BE%D1%81%D0%B8%D1%82%D1%8C-%D0%BF%D1%80%D0%BE%D0%BC%D0%BE%D0%BA%D0%BE%D0%B4-%D1%83-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8F-admitad_1).


Если вы не размещаете персональные промокоды, раздел будет пустым. 

## Экспорты


Если вы хотите организовать собственную аналитику на основании нашей статистики, воспользуйтесь опцией «Экспорт» справа от фильтров.


Экспортированные файлы попадают на вкладку «Экспорт» и доступны к скачиванию в течение 90 дней.
