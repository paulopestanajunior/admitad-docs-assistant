# Как добавить описание для партнерской программы в Admitad

Описание вашей программы — один из ключевых факторов, на основании которого веб\-мастера принимают решение о сотрудничестве с ней.


Описание вашей программы отображается веб\-мастерам:

- в карточке программы в каталоге [Admitad Store](https://store.admitad.com/ru/catalog/);
- на странице программы.

*В каталоге для неавторизованных пользователей информация о вашей программе обновляется раз в сутки.*

## Порядок заполнения информации о программе


Порядок заполнения зависит от типа управления программой. 

- Управление через аккаунт\-менеджера — в этом случае обратитесь к своему менеджеру, и он добавит или отредактирует информацию в вашей программе.
- Самостоятельное управление — в этом случае вы добавляете информацию о программе самостоятельно.


Чтобы самостоятельно добавить или изменить информацию о программе, перейдите в раздел «Информация о программе» (Программа → Профиль программы → Информация о программе).


Ниже вы найдете рекомендации по заполнению всех полей в этом разделе.


После заполнения всех полей в разделе, нажмите «Предварительный просмотр» и проверьте, как веб\-мастера будут видеть страницу вашей программы.

## Название


В поле «Название» укажите название вашей программы или бренда. Под ним ваша программа будет отображаться веб\-мастерам в каталоге программ [Admitad Store](https://store.admitad.com/ru/catalog/).


Рекомендации:

- Укажите название, которое точно совпадает с названием вашего бренда. Так веб\-мастера легко смогут найти программу по названию с помощью поиска.
- Не добавляйте краткое описание проекта в название программы.
	- Неправильно: «Sindbad — дешевые авиабилеты онлайн».
	- Правильно: «Sindbad».
- Не добавляйте в название программы доменное имя. Например «ru» или «com».
	- Неправильно: «Aliexpress.com».
	- Правильно: «Aliexpress».


Также в названии вы можете указать [регион программы](#geo-in-program-title) и [модель оплаты](#postfixes-with-additional-information), по которой работает программа. По этим обозначениям веб\-мастера могут быстро определить, подходит ли для них ваша программа.

- В названии вы можете указать регион(ы) — страны, с которыми работает ваша программа. Такое обозначение поможет веб\-мастерам понять, трафик из каких регионов разрешен в программе и стоит ли им подключаться. 

| Регионы в программе | Формат обозначения в названии | Пример |
| --- | --- | --- |
| 1 регион | Укажите 1 двухбуквенный код региона | Romwe US |
| 2–5 регионов | Укажите двухбуквенные коды регионов | Perfumes Club ES DE IT FR |
| Программа работает более чем с 5 регионами | Укажите *Many GEOs* | SHEIN Many GEOs |
| Программа работает с регионами по всему миру | Укажите *WW\** | Rentalcars WW |


\* Чтобы использовать обозначение «WW», ваш сайт или другой проект должен быть доступен на английском языке (помимо локального). Также в этом случае необходимо добавить описание и правила программы на английском языке (помимо локального).

Дополнительные требования к использованию обозначений:

	- Если вы запускаете несколько отдельных программ для разных регионов, то в названии каждой укажите двухбуквенный код соответствующего региона.
	- Если в программе возможны заказы только из одного или нескольких городов, добавьте в название обозначение страны, а в [правилах программы](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B4%D0%BE%D0%B1%D0%B0%D0%B2%D0%B8%D1%82%D1%8C-%D0%BF%D1%80%D0%B0%D0%B2%D0%B8%D0%BB%D0%B0-%D0%BF%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%BE%D0%B9-%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D1%8B-%D0%B2-admitad_1) укажите города.
- Постфиксы актуальны для программ из категорий «Онлайн\-игры», «Мобильные программы», «Финансовые программы».


В названии программы вы также можете указать дополнительную информацию о ней: 

	- модель оплаты — см. список сокращений ниже;
	- и ОС вашего приложения — iOS или Android.
Если одновременно с постфиксом вы используете обозначение региона, укажите его в формате: *Название \[постфикс] {код региона}.*

Список разрешенных сокращений:

| Сокращение | Расшифровка |
| --- | --- |
| SOI | Single\-opt\-in |
| DOI | Double\-opt\-in |
| CPP | Cost per player |
| CPS | Cost per sale |
| CPC | Cost per click |
| CPL | Cost per lead |
| CPV | Cost per visit |
| CPI | Cost per Install |
| CPA | Cost per action.*Используйте это сокращение для общего обозначения Cost per mailer, Cost per registration on app и других моделей, не представленных в списке. Уточните используемую модель в описании программы.* |
| API | Application programming interface |
| APK | Android package kit |

  
Другие требования к использованию постфиксов:

	- Если в программе оплачиваются несколько типов действий:
	- - укажите все постфиксы через запятую в квадратных скобках  
		*Пример: Название \[CPC, CPL, CPS] RU*
		- либо укажите только основной  
		*Пример: CPR и SOI — Название \[SOI] DE*
	- Если в программе оплачивается одно из нескольких действий, укажите постфиксы через слэш.  
	*Пример: Name \[CPL/CPS] UA*
> *Пример 1 — Banggood \[CPI, Android] DE*
> 
> 
> 
> 	- *CPI (Cost Per Install) — оплачивается установка приложения;*
> 	- *Android — приложение для Android;*
> 	- *DE — программа работает только в Германии.*
> *Пример 2 — Star Stable \[SOI] DE AT CH*
> 
> 
> 
> 	- *SOI — оплачиваются регистрации, которые не требуют подтверждения.*
> 	- *DE AT CH — программа работает только в Германии, Австрии, Швейцарии.*

## Ссылка на сайт


В поле «Ссылка на сайт» проверьте, что ссылка на ваш сайт указана верно.


Ссылка не должна содержать дополнительных меток или параметров.


По этой ссылке веб\-мастера смогут перейти на ваш сайт и ознакомиться с продуктами и услугами, которые им нужно будет продвигать и рекламировать.

## Варианты написания названия


В поле «Варианты написания названия» через запятую укажите возможные варианты написания названия вашей программы.


Перечислите варианты на транслите, с наиболее частыми опечатками, на других раскладках клавиатуры и пр. 


Это поможет веб\-мастеру найти вашу программу в Admitad Store, даже если он напишет название бренда неверно.

## Логотип


Загрузите логотип вашей компании в формате SVG.

## Описание


Информация в поле «Описание» — один из ключевых факторов, которыми руководствуются веб\-мастерам при выборе программы для сотрудничества. 


Важно составить привлекательное описание вашего магазина или сервиса — так вы сможете заинтересовать веб\-мастеров и мотивировать их подключиться к вашей программе.

Кроме того, информацию из описания (портрет аудитории, топ\-товары) веб\-мастера будут использовать для привлечения пользователей на ваш сайт.


Описание всегда должно быть актуальным. Если вы хотите изменить описание программы, отредактируйте их самостоятельно или попросите об этом своего менеджера.


Если вы добавляете информацию о программе впервые, вы можете использовать шаблон описания, представленный в поле.

Заполните описание по шаблону:

- Краткое описание вашей компании — должно содержать сведения о направлении деятельности, включая информацию об основных продуктах и географию продаж. Объем: не более 3–4 строк.
- Минус\-слова — перечислите слова и словосочетания, которые веб\-мастера не должны использовать в контекстных рекламных объявлениях (например, название бренда), если такие есть.
- Преимущества для клиентов — укажите преимущества, которые отличают вас от конкурентов — например, круглосуточная бесплатная доставка или эксклюзивные бренды.
- Портрет целевой аудитории — описание представителя вашей целевой аудитории: пол, возраст, доход, профессия, семейное положения и т.д.
- Популярные товары — перечислите категории или продукты, которые продаются лучше всего.
- Рекомендации по позиционированию бренда — посоветуйте веб\-мастерам, на чем стоит сделать акцент; есть ли слова или выражения, которые нужно использовать в рекламе и т.д.


Если ваша программа относится к категории «Онлайн\-игры» «Мобильные программы» или «Финансовые программы», добавьте в описание информацию, указанную ниже.

- - Онлайн\-игры:
		- KPI, по которым будет оцениваться трафик;
		- тестовые лимиты;
		- ссылка на хранилище с креативами и правила их использования.
	- Мобильные программы:
		- KPI, по которым будет оцениваться трафик.
	- Финансовые программы
		- копия лицензии банка.

## Краткое описание


В поле «Краткое описание» добавьте сокращенное описание своей программы. Объем: не более 1–2 строк.


Краткое описание используется в общей рассылке веб\-мастерам о новых программах.


Вы можете просто скопировать 1–2 предложения из поля «Описание».

## Что делать после заполнения описания программы


После заполнения всех полей:


1\. Нажмите «Предварительный просмотр», чтобы посмотреть, как веб\-мастера будут видеть описание на странице вашей программы.


2\. Нажмите «Сохранить», чтобы сохранить изменения и завершить редактирование.


Готово. Описание вашей программы появится (обновится) на странице программы и в карточке программы в каталоге [Admitad Store](https://store.admitad.com/ru/catalog/).
