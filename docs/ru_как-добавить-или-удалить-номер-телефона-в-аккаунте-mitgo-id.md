# Как добавить или удалить номер телефона в аккаунте Mitgo ID

Номер телефона может использоваться, чтобы подтвердить, что именно вы являетесь владельцем аккаунта.


Если вы утратите доступ к email, на который зарегистрирован аккаунт, номер телефона поможет ускорить процесс восстановления доступа.


В аккаунте Mitgo ID вы можете:

- [добавить номер телефона](#adding-phone-number);
- [удалить текущий номер телефона](#deleting-phone-number).


Чтобы изменить номер телефона, сначала удалите текущий номер, а затем добавьте новый.

## Как добавить номер телефона в аккаунте Mitgo ID


1\. В разделе [«Безопасность»](https://id.mitgo.com/security) нажмите на поле «Номер телефона» в блоке «Email и номер телефона».


2\. На ваш email будет отправлено письмо с кодом для подтверждения запроса на добавление номера телефона.  
[Что делать, если письмо не пришло](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B8%D0%B7%D0%BC%D0%B5%D0%BD%D0%B8%D1%82%D1%8C-email-%D0%B2-%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B5-mitgo-id_1#did-not-receive-confirmation-email)


3\. В аккаунте Mitgo ID на открывшейся странице введите код из полученного письма и нажмите «Подтвердить».


4\. На открывшейся странице введите номер телефона, который вы хотите добавить, и нажмите «Далее».


5\. На указанный номер будет отправлено СМС с кодом для подтверждения номера телефона.


Введите код из полученного СМС и нажмите «Подтвердить».

Готово! Вы добавили номер телефона в свой аккаунт Mitgo ID.


Если вы не получили СМС с кодом, нажмите «Начните процесс заново».

## Как удалить номер телефона в аккаунте Mitgo ID


1\. В разделе [«Безопасность»](https://id.mitgo.com/security) нажмите на поле «Номер телефона» в блоке «Email и номер телефона».


2\. На открывшейся странице нажмите «Удалить номер».


На ваш email будет отправлено письмо с кодом для подтверждения запроса на удаление номера телефона.  
[Что делать, если письмо не пришло](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B8%D0%B7%D0%BC%D0%B5%D0%BD%D0%B8%D1%82%D1%8C-email-%D0%B2-%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B5-mitgo-id_1#did-not-receive-confirmation-email)


3\. В аккаунте Mitgo ID на открывшейся странице введите код из полученного письма и нажмите «Подтвердить».

Готово! Ваш номер телефона успешно удален из аккаунта Mitgo ID.

*Чтобы добавить новый номер телефона, нажмите «Добавить номер телефона» и следуйте [инструкции выше](#adding-phone-number).*
