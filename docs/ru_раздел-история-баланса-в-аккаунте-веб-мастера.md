# Раздел «История баланса» в аккаунте веб-мастера

В разделе «[История баланса](https://store.admitad.com/ru/webmaster/balance-history/)» (Финансы → История баланса) вы можете посмотреть полную историю движения средств по вашему балансу в Admitad.


Данные представлены в виде таблицы со списком операций, которые отсортированы по дате от недавних к более ранним. [Подробнее](#data-in-history)

## Как найти операцию в разделе «История баланса»


Чтобы быстро найти нужные операции в истории баланса, используйте фильтр над таблицей.


Поиск работает как по отдельным параметрам, так и по нескольким одновременно. Список параметров, которые доступны при поиске:

- по дате операции;
- по валюте;
- по типу операции;
- по причине;
- по ID целевого действия. ID всех действий доступны в разделе «[Статистика по действиям](https://store.admitad.com/ru/webmaster/statistics/actions/)».

## Какие данные доступны в разделе


Таблица содержит следующие колонки:

- Дата — дата проведения операции.
- Сумма — сумма, которая была начислена или списана с баланса.
- Валюта — валюта начисленной или списанной суммы.
- Тип операции:
	- Пополнение — поступление средств на баланс.  
	Например, начисление вознаграждения за целевое действие, возврат средств по заявке на вывод, если ее не удалось выполнить, или начисление после ошибочного списания.
	- Снятие — списание с баланса.  
	Например, в результате перевода средств в [Admitad Earnings Wallet](https://support.admitad.ru/knowledge-base/article/%D1%87%D1%82%D0%BE-%D1%82%D0%B0%D0%BA%D0%BE%D0%B5-mitgo-wallet_1). Или списание вознаграждения за целевое действие, которое рекламодатель отклонил после подтверждения. [Подробнее об отклонении уже подтвержденных действий](https://support.admitad.ru/knowledge-base/article/%D0%BF%D0%BE%D1%87%D0%B5%D0%BC%D1%83-%D1%83%D0%BC%D0%B5%D0%BD%D1%8C%D1%88%D0%B8%D0%BB%D0%B0%D1%81%D1%8C-%D1%81%D1%83%D0%BC%D0%BC%D0%B0-%D0%B2-%D1%81%D1%82%D0%B0%D1%82%D1%83%D1%81%D0%B5-%C2%AB%D0%B3%D0%BE%D1%82%D0%BE%D0%B2%D1%8B-%D0%BA-%D1%81%D0%BD%D1%8F%D1%82%D0%B8%D1%8E%C2%BB-%D0%B8%D0%BB%D0%B8-%C2%AB%D0%BE%D0%B6%D0%B8%D0%B4%D0%B0%D1%8E%D1%82-%D0%BE%D0%BF%D0%BB%D0%B0%D1%82%D1%8B-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D0%B5%D0%BC%C2%BB_1)


Списания в «Истории баланса» отображаются только по действиям, которые получили статус «Подтвержден». Например, если в процессе сверки были отклонены действия в статусе «Ожидают» или «Подтвержден, ожидает оплаты рекламодателем», то вознаграждения за такие действия просто исчезнут с вашего баланса. При этом в «Истории баланса» списания по таким действиям не отобразятся. Вы можете найти информацию по ним в разделе «Статистика по действиям», отфильтровав действия по статусу «Отклонен».

- Причина — пояснение, почему средства были списаны или начислены.
	- Платеж по действию ID\*\*\* — означает, что рекламодатель подтвердил действие с ID\*\*\*, и на баланс было зачислено вознаграждение за него.  
	Если позже рекламодатель отклонит подтвержденное ранее действие, то в истории баланса отобразится еще одна операция с причиной «Платеж по действию ID\*\*\*», но на этот раз с типом «снятие». [Подробнее об отклонении уже подтвержденных действий](https://support.admitad.ru/knowledge-base/article/%D0%BF%D0%BE%D1%87%D0%B5%D0%BC%D1%83-%D1%83%D0%BC%D0%B5%D0%BD%D1%8C%D1%88%D0%B8%D0%BB%D0%B0%D1%81%D1%8C-%D1%81%D1%83%D0%BC%D0%BC%D0%B0-%D0%B2-%D1%81%D1%82%D0%B0%D1%82%D1%83%D1%81%D0%B5-%C2%AB%D0%B3%D0%BE%D1%82%D0%BE%D0%B2%D1%8B-%D0%BA-%D1%81%D0%BD%D1%8F%D1%82%D0%B8%D1%8E%C2%BB-%D0%B8%D0%BB%D0%B8-%C2%AB%D0%BE%D0%B6%D0%B8%D0%B4%D0%B0%D1%8E%D1%82-%D0%BE%D0%BF%D0%BB%D0%B0%D1%82%D1%8B-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D0%B5%D0%BC%C2%BB_1)  
	
	
	Кликните по тексту причины, чтобы перейти в статистику и посмотреть подробную информацию по этому действию: к какой программе и площадке оно относится, тариф, сумму заказа и т.д.
	- Заявка на выплату — означает, что вы перевели средства в Admitad Earnings Wallet и указанная сумма была списана с вашего баланса в Admitad.
	- Списание за фродовые действия — эта причина означает, что некоторые вознаграждения, которые вы уже успели вывести из Admitad, были списаны с вашего баланса и возвращены на баланс рекламодателя, т.к. он отклонил часть подтвержденных ранее действий за нарушения правил программы. [Подробнее](https://support.admitad.ru/knowledge-base/article/%D0%BF%D0%BE%D1%87%D0%B5%D0%BC%D1%83-%D1%83%D0%BC%D0%B5%D0%BD%D1%8C%D1%88%D0%B8%D0%BB%D0%B0%D1%81%D1%8C-%D1%81%D1%83%D0%BC%D0%BC%D0%B0-%D0%B2-%D1%81%D1%82%D0%B0%D1%82%D1%83%D1%81%D0%B5-%C2%AB%D0%B3%D0%BE%D1%82%D0%BE%D0%B2%D1%8B-%D0%BA-%D1%81%D0%BD%D1%8F%D1%82%D0%B8%D1%8E%C2%BB-%D0%B8%D0%BB%D0%B8-%C2%AB%D0%BE%D0%B6%D0%B8%D0%B4%D0%B0%D1%8E%D1%82-%D0%BE%D0%BF%D0%BB%D0%B0%D1%82%D1%8B-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D0%B5%D0%BC%C2%BB_1)
		- В комментарии к причине указывается программа, к которой относится списание.
		- Чтобы посмотреть детальную информацию по списанию, нажмите «Подробнее». Откроется поп\-ап, в котором вы можете скачать подробный отчет по всем отклоненным действиям.
	- Транзакция по балансам — эта причина используется для ручных операций по балансу, а также для начисления вознаграждения по тем заявкам из инструмента «Потерянные заказы», которые были подтверждены Admitad без участия рекламодателя. Такая транзакция будет с комментарием: «Программа {название программы}. Потерянный заказ №{номер заказа}. Подтвержден Admitad».
	- Перевод средств между веб\-мастерами — используется для обозначения операций по переводу средств с вашего баланса Admitad на баланс в [GetUniq](https://support.admitad.ru/knowledge-base/article/getuniq_2).
	- Реферальный платеж — означает начисление средств согласно условиям реферальной программы Admitad. [Подробнее о реферальной программе](https://support.admitad.ru/knowledge-base/article/%D0%B5%D1%81%D1%82%D1%8C-%D0%BB%D0%B8-%D1%83-%D0%B2%D0%B0%D1%81-%D1%80%D0%B5%D1%84%D0%B5%D1%80%D0%B0%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F-%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B0_1)
	- Instant Payout Lite — используется для транзакций по балансу, которые связаны с заявками в [сервис Instant Payout Lite](https://support.admitad.ru/knowledge-base/article/admitad-instant-payout_6).
	
	Причина может содержать следующие комментарии:
	
	
	
		- *Заявка Instant Lite Payout ID\*\*\** — означает, что средства переведены в статус «Готовы к снятию» в результате выполнения заявки в Instant Payout Lite.
		- *Instant Payout Lite fee по заявке ID\*\*\** — означает списание суммы уменьшения вознаграждения в результате выполнения заявки в Instant Payout Lite.
		- *Корректировка баланса за ранее выплаченное вознаграждение по действию ID\*\*\* из заявки Instant Payout Lite ID \*\*\*\** — означает списание в рамках корректировки баланса: рекламодатель оплатил подтвержденные действия, и теперь система списывает эту сумму, т.к. ранее эти средства уже были начислены на баланс в рамках заявки в Instant Payout Lite.
	- Instant Payout Pro — используется для транзакций по балансу, которые связаны с заявками в [сервис Instant Payout Pro](https://support.admitad.ru/knowledge-base/article/admitad-instant-payout_6#instant-payout-pro).
	
	Причина может содержать следующие комментарии:
	
	
	
		- *Заявка Instant Payout Pro ID\*\*\** — означает, что средства переведены в статус «Готовы к снятию» в результате выполнения заявки в Instant Payout Pro.
		- *Корректировка баланса за ранее выплаченное вознаграждение по действию ID\*\*\* из заявки Instant Payout Pro ID \*\*\*\** — означает списание в рамках корректировки баланса: рекламодатель оплатил подтвержденные действия, и теперь система списывает эту сумму, т.к. ранее эти средства уже были начислены на баланс в рамках заявки в Instant Payout Pro.
		- *Компенсация по заявке в Instant Payout Pro\*\*\** — это доначисление вознаграждений за действия из заявки в Instant Payout Pro, которые рекламодатель подтвердил и оплатил. Компенсация означает, что предсказанная системой сумма вознаграждений оказалась меньше, чем итоговая, и Admitad начислил разницу на ваш баланс. [Подробнее о компенсации по заявке](https://support.admitad.ru/knowledge-base/article/admitad-instant-payout_6#recalculation-pro)
- Комментарий — дополнительная информация об операции.
