# Take Link overview

Take Link is a Takeads product that helps publishers profit off of their platforms, such as content websites, blogs, and forums.

## How Take Link works

1\. Take Link analyzes content on your website's pages and finds direct links leading to the websites of advertisers from the Takeads catalog.

*An advertiser is a brand (e.g., a store, bank, online school, etc.) that seeks to boost sales and is ready to pay platform owners for leading traffic to advertisers' websites and encouraging users to perform a target action (e.g., purchase a product, place an order, or install an app).*

2\. Take Link automatically processes such links by adding special tags to them to track user actions and clicks. These 'converted' links are called *affiliate links*.

3\. The platform owner receives a reward if the following conditions are met:

- The platform user came to the advertiser's website through an affiliate link and placed an order or bought something.
- The advertiser checked and confirmed that the order was made and did not violate the rules.

## Take Link integration methods

Currently, there are several integration methods available:

- Add JavaScript code to your website's source ([guide](https://admitad.useresponse.com/knowledge-base/article/how-to-add-a-platform-on-takeads_5#copy-js-code)).
- Use the WordPress plugin if your website is built with WordPress ([guide](https://support.admitad.com/hc/en-us/articles/11452082622993)).

To integrate Take Link with JavaScript, you need to have access to the source code of the website you use as a platform in Takeads. If your website is built on a CMS, you can deploy the code through the website template.
