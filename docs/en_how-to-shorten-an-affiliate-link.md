# How to shorten an affiliate link?

If you want to know, how to get an affiliate link, please read [this article](https://support.mitgo.com/hc/en-us/articles/360019067357).

To shorten an affiliate link, use the Shortlink tool (Tools — Shortlink). Just paste your link in the field and click Get a short link.

Note: This tool shortens only Admitad affiliate links.
