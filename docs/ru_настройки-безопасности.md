# Настройки безопасности

Включите двухфакторную аутентификацию для подтверждения платежных данных в аккаунте — это поможет максимально обезопасить заработанное. 


Двухфакторная аутентификация — это дополнительный уровень защиты вашего аккаунта, который гарантирует, что доступ к нему сможете получить только вы, даже если ваш пароль стал известен кому\-то еще.

## Как настроить двухфакторную аутентификацию


Если вы уже настроили двухфакторную аутентификацию [в своем аккаунте Mitgo ID](https://id.mitgo.com/security/totp), то дополнительно включать ее в Mitgo Wallet не нужно.

1. В разделе [«Платежные данные»](https://earnings.admitad.com/billing-details?_gl=1*1n74zam*_ga*OTk0MTc3NjIxLjE2ODk3NjY2MTE.*_ga_32HHLCD33P*MTcwNjg2NjUzNS44LjEuMTcwNjg2Njg1NC41MC4wLjA.) нажмите кнопку «Включить» в блоке «Включите двухфакторную аутентификацию».
2. В открывшемся окне нажмите «Отправить письмо с кодом». Код придет на вашу электронную почту. Повторно запросить письмо с кодом можно не раньше, чем через минуту.
3. Введите полученный код и нажмите «Подтвердить».
4. Откроется страница настройки — следуйте подсказкам в интерфейсе:


4\.1 скачайте приложение Google Authenticator на свое устройство:

- [ссылка для iOS](https://apps.apple.com/ru/app/google-authenticator/id388497605);
- [ссылка для Android](https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2).


4\.2 отсканируйте QR\-код на экране — ваш аккаунт будет добавлен в приложение Google Authenticator, и вы увидите 6\-значный код, который обновляется каждые 30 секунд;


4\.3 сохраните ключ для сброса — без него вы не сможете сбросить двухфакторную аутентификацию и восстановить доступ к аккаунту, если утратите доступ к устройству;


4\.4 введите этот код в поле «Код из приложения»;


4\.5 укажите название устройства, на котором установлено приложение, в поле «Название устройства».


5\. Нажмите «Включить защиту».


Готово, вы настроили двухфакторную аутентификацию. Теперь для подтверждения новых платежных данных вам нужно будет вводить одноразовый код из приложения Google Authenticator.
