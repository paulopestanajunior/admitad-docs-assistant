# Порядок вывода денег

Если вы выводите деньги из Wallet первый раз, последовательно выполните все шаги по этим инструкциям:

1. [Добавьте в Wallet налоговую информацию](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B4%D0%BE%D0%B1%D0%B0%D0%B2%D0%B8%D1%82%D1%8C-%D0%BD%D0%B0%D0%BB%D0%BE%D0%B3%D0%BE%D0%B2%D1%83%D1%8E-%D0%B8%D0%BD%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%86%D0%B8%D1%8E_1).
2. [Добавьте в Wallet платежные данные](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B4%D0%BE%D0%B1%D0%B0%D0%B2%D0%B8%D1%82%D1%8C-%D0%BF%D0%BB%D0%B0%D1%82%D0%B5%D0%B6%D0%BD%D1%8B%D0%B5-%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D0%B5_1). Для этого у вас должна быть настроена [двухфакторная аутентификация](https://support.admitad.ru/knowledge-base/article/%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B8-%D0%B1%D0%B5%D0%B7%D0%BE%D0%BF%D0%B0%D1%81%D0%BD%D0%BE%D1%81%D1%82%D0%B8_1).
3. Переведите деньги из личного кабинета в Wallet:
	- [Инструкция для Admitad](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B2%D1%8B%D0%B2%D0%B5%D1%81%D1%82%D0%B8-%D1%81%D1%80%D0%B5%D0%B4%D1%81%D1%82%D0%B2%D0%B0-%D0%B8%D0%B7-admitad_1)
	- [Инструкция для Takprodam](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B2%D1%8B%D0%B2%D0%B5%D1%81%D1%82%D0%B8-%D0%B4%D0%B5%D0%BD%D1%8C%D0%B3%D0%B8-%D0%B8%D0%B7-takprodam)
	- [Инструкция для Affilead](https://support.admitad.ru/knowledge-base/article/faq-%D0%BF%D0%BE-%D0%B2%D1%8B%D0%B2%D0%BE%D0%B4%D1%83-%D1%81%D1%80%D0%B5%D0%B4%D1%81%D1%82%D0%B2-%D0%B8%D0%B7-affilead)
4. [Создайте заявку на перевод денег.](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D1%81%D0%BE%D0%B7%D0%B4%D0%B0%D1%82%D1%8C-%D0%B7%D0%B0%D1%8F%D0%B2%D0%BA%D1%83-%D0%BD%D0%B0-%D0%B2%D1%8B%D0%B2%D0%BE%D0%B4-%D0%B4%D0%B5%D0%BD%D0%B5%D0%B3_1)


В следующий раз вывести деньги будет быстрее и проще, так как вы сможете использовать уже указанную налоговую информацию и платежные данные.
