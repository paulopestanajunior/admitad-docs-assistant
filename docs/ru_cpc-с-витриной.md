# CPC с витриной

«CPC с витриной» — это модель оплаты рекламы, при которой продавец платит вам за уникальные переходы реальных пользователей с витрины на товар маркетплейса. Все продавцы, кроме AliExpress, оплачивают такие клики по фиксированной ставке. Получить ссылку на витрину и ознакомиться с потенциальной комиссией за продвижение можно в разделе «Каталог товаров» → «Промо\-витрины». Подробнее о [витрине](https://support.admitad.ru/knowledge-base/article/%D1%87%D1%82%D0%BE-%D1%82%D0%B0%D0%BA%D0%BE%D0%B5-%D0%BF%D1%80%D0%BE%D0%BC%D0%BE-%D0%B2%D0%B8%D1%82%D1%80%D0%B8%D0%BD%D0%B0)


У каждой кампании есть лимит выплат — максимальная сумма, которую может получить паблишер за продвижение товаров витрины кампании. При достижении лимита паблишер больше не получает комиссии за переходы в рамках этой кампании и может выбрать витрину другой кампании для продвижения.

# Как AliExpress оплачивает CPC\-комиссию


Продавец платит вам за уникальные клики реальных пользователей с учетом конверсии трафика. Чем больше заказов и переходов с витрины в карточку товара, тем выше ваша комиссия:   
Комиссия \= цена клика × количество переходов × коэффициент эффективности.


Коэффициент эффективности — процентное отношение количества продаж по ссылке к общему числу переходов (кликов) по партнерской ссылке.


Если по вашей партнерской ссылке есть переходы с витрины в карточку товара, но нет продаж — коэффициент будет равен нулю. При качественном трафике, который приносит продажи, фактическая цена клика может в 1,5 раза превышать установленную продавцом цену клика.


Посмотреть информацию о комиссии за продвижение можно в разделе «Статистика».

# Как продвигать отдельный товар с витрины


Продвигать по CPC можно не только всю витрину кампании, но и отдельные товары. Для каждого товара витрины есть свой лендинг — посадочная страница с основными характеристиками товара и ссылкой на товар на маркетплейсе. Оплата осуществляется за переход по ссылке с лендинга в карточку на маркетплейсе.


Продвижение товаров витрины кампании ограничено лимитом выплат на одного паблишера. При достижении лимита комиссии не выплачиваются.


Чтобы получить ссылку на лендинг товара:


1\. Перейдите в раздел «Каталог товаров».


2\. В поле «Площадка» выберите площадку, на которой хотите продвигать товары витрины.


Если нет доступных для выбора площадок, добавьте и подтвердите площадку по [инструкции](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%BD%D0%B0%D1%87%D0%B0%D1%82%D1%8C-%D1%80%D0%B0%D0%B1%D0%BE%D1%82%D1%83-%D1%81-%D1%82%D0%B0%D0%BA%D0%BF%D1%80%D0%BE%D0%B4%D0%B0%D0%BC_1).


3\. В фильтре «Модель оплаты» выберите «CPC с витриной».


4\. В списке товаров выберите товар, который хотите продвигать, и скопируйте партнерскую ссылку на его лендинг.


5\. Разместите партнерскую ссылку у себя на площадке, добавив в сопровождающий креатив данные о рекламодателе.
