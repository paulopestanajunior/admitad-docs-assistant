# How to join an affiliate program?

Once you [signed up](https://support.admitad.com/hc/en-us/articles/360019927878) to Admitad and [added an ad space](https://support.admitad.com/hc/en-us/articles/360019060997), you need to join an [affiliate program](https://support.admitad.com/hc/en-us/articles/4403304880529#affiliate-program-en).


We recommend that you read the related texts:

- [Overview of the affiliate program catalog](https://support.admitad.com/hc/en-us/articles/360019182618)
- [Affiliate programs for beginners](https://admitad.academy/en/affiliate-programs-for-the-beginners/)
- [How to choose an affiliate program?](https://support.admitad.com/hc/en-us/articles/360019386758)


You can join an affiliate program:

- [in the Admitad Store catalog](#join-in-catalog)
- [on the affiliate program page](#join-on-page)

## How to join a program from the catalog


Go to the [catalog](https://store.admitad.com/en/catalog/) and pick a program you would like to join.


Use the filters on the left to quickly find programs that offer the terms you need.  
For instance, to find programs you could join instantly, check the Auto moderation box in the filters. [Learn more about filters here](https://help.admitad.com/en/topic/289-overview-of-the-affiliate-program-catalog#filters)


If the Join button is inactive in the program card, you won't be able to connect your ad space to it. [Learn more about the reasons](https://support.admitad.com/hc/ru/articles/4409918037777)


If you see the violet Join button in the program card, it means the program allows autoconnection.  
Click Join, read the terms and, if you agree with them, click Accept.

  
If your ad space complies with the rules and terms, it will be connected to the program instantly.  
Once joined, you can start work: get an affiliate link and deploy it in your ad space. [Learn more about ways to get a link](https://support.admitad.com/hc/en-us/articles/360019067357)


If you see the green Join button in the program card, this means the program is set up for manual moderation by the advertiser. To join a program, click Apply for moderation, read the program rules, and click Accept.


Please note: Some programs issue rewards only in a specific currency. This information is always stated in the program rules. By accepting the rules, you confirm that you are able to receive payments in the specified currency and by the specified method.

## How to join on the program page


Go to the [catalog](https://store.admitad.com/en/catalog/) and choose a program you want to join.  
Use the filters on the left to quickly find programs that offer the terms you need.  
For instance, to find programs you could join instantly, check the Auto moderation box in the filters. [Learn more about filters here](https://support.admitad.com/hc/en-us/articles/360019182618-Overview-of-the-affiliate-program-catalog#filters)


To open the program page, click its logo (1\), name (2\), or the Learn more button (3\).

  
 


If instead of the Join button on the program page you see the hint "You can't join this program", you won't be able to connect your ad space to it. [Learn more about the reasons](https://support.admitad.com/hc/ru/articles/4409918037777)


If you see the Join button on the program page, it means the program allows autoconnection. Click Join.

  
 


In the new window, read the program rules, check the box if you agree with them, and click Join again. If your ad space complies with the rules and terms, it will be connected to the program instantly. Once joined, you can start work: get an affiliate link and deploy it in your ad space. [Learn more about ways to get a link](https://support.admitad.com/hc/en-us/articles/360019067357)


If you see the Apply button on the program page, this means the program is set up for manual moderation by the advertiser. To join the program, click Apply.


You'll be able to view the status of the join request on the program card. The status reflects the stage of the moderation process.


Please note: Some programs issue rewards only in a specific currency. This information is always stated in the program rules. By accepting the rules, you confirm that you are able to receive payments in the specified currency and by the specified method.

## Moderation statuses


In the new window, read the program rules, check the box if you agree with them, and click Join again. Your join request will be submitted to the advertiser; they have to review it within 5 days. You can learn when moderation will be finished in the program card in the catalog.

  
If your ad space meets the advertiser's requirements, they will connect your space to the program; you will receive a confirmation email. Once joined, you can start work: get an affiliate link and deploy it in your ad space. [Learn more about ways to get a link](https://support.admitad.com/hc/en-us/articles/360019067357)


If your ad space does not comply with the program rules, the advertiser may decline your request. In this case, you will get a declination email. [Learn more about reasons for declining](https://support.admitad.com/hc/en-us/articles/360019062817)


To learn why the advertiser declined your request, click Find out why in the program card.


If the review period is expired but you received no response from the advertiser, you will see a notification on the program card.

  
For one, moderation may drag on if the advertiser is overwhelmed with requests and they just cannot make it in time to process all of them. [Learn more about moderation periods](https://support.admitad.com/hc/en-us/articles/360019067057).  
In this case, you can [create a request for Admitad support](https://support.admitad.com/hc/en-us/articles/360020421978) and describe how you are going to promote the advertiser's goods or services. This way, you will be able to draw the advertiser's attention to your request and they will make a resolution faster. To create a ticket, click Complain.

## How to find programs you have already joined


To find programs you have already joined, mark Joined in the Status filter.  

## After you have joined a program


Now that you are connected to the program, you need to get an affiliate link and publish it on your ad space.
