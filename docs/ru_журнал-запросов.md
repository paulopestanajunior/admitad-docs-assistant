# Журнал запросов

Раздел представляет собой журнал запросов с данными о действиях, которые были совершены в вашей программе.

На этапе интеграции здесь можно посмотреть, корректно ли передались данные о тестовых заказах с вашей стороны на сторону Admitad, а после начала работы программы — выявить ошибки, если часть заказов вдруг перестала попадать в статистику или загрузилась с неверными данными. 

Для каждого запроса указано:

- время отправки запроса;
- текст самого запроса;
- текст ошибки.

Если в поле «Результат» стоит прочерк, значит запрос передан корректно.

 

Ниже дано описание ошибок, которые вызваны неправильными настройками трекинга и должны быть исправлены.

| Текст ошибки | Описание ошибки | Решение |
| --- | --- | --- |
| UIDDoesNotExist | admitad\_uid отсутствует или имеет некорректное значение. | Проверить admitad\_uid, исправить или добавить значение в запрос. |
| ActionCodeDoesNotExist | В запросе отсутствует код действия (action\_code, ac). | Добавить action\_code в запрос. |
| ActionDoesNotExist | Неверное значение action\_code (ac). | Исправить значение в запросе. |
| TariffDoesNotExist | В запросе отсутствует tariff\_code (также «tc»), или у него неправильное значение. | Исправить или добавить параметр tariff\_code. |
| PaymentExistError | Действие с таким order\_id (oid) уже есть в системе. | Проверить order\_id. |
| PixelTrackingNotAllowed | Pixel\-запросы не разрешены. Ошибка встречается у программ, которые интегрировались через трекинг\-код. | Если программа сначала была интегрирована через трекинг\-код, а потом перешла на другой тип интеграции, ошибку можно игнорировать.Если тип интеграции — трекинг\-код, свяжитесь со специалистом отдела трекинга, он поправит настройки. |
| PostbackNotAllowed | Ошибка аналогична предыдущей, но возникает у программ с интеграцией через postback\-запросы. | Если программа сначала была интегрирована через postback, а потом перешла на другой тип интеграции, ошибку можно игнорировать.Если тип интеграции — postback, свяжитесь со специалистом отдела трекинга, он поправит настройки. |
| PostbackAuthorizationError | Ошибка аутентификации postback\-запроса. Ошибка может быть вызвана некорректным значением или вообще отсутствием postback\_key. Также система выдаст ошибку, если у вас есть список разрешенных IP\-адресов, с которых можно отправлять postback\-запросы, а адрес этого запроса не входит в список. | Проверить наличие и корректность postback\_key. |
| PaymentAlreadyCollected | Запрос с информацией об этом действии уже был получен. Ошибка встречается только у действий типа «продажа» (payment\_type\=sale) | Если заказ есть в статистике и данные верны, править ничего не нужно. В ином случае проверьте сумму заказа, position\_id и position\_count. |
| FingerprintError | Ошибка аналогична предыдущей, но свойственна для типа действия «лид» (payment\_type\=lead). | Проверить статистику. Если лид загружен, данные верны, то править ничего не нужно. |

 

После исправления ошибки не забудьте сделать тест, чтобы убедиться, что теперь все данные передаются корректно.  
 

Далее представлены ошибки, которые вызваны не неправильным трекингом, а иными причинами. Их править не нужно.

| Текст ошибки | Описание |
| --- | --- |
| RateDoesNotExist | Нет ставки.Ошибка встречается чаще всего у программ, где настроены ставки по регионам. Показывает, что запрос пришел из страны, для которой ставка не определена. |
| CookiesIsExpired | Истекло время жизни cookie. Далее указано, насколько давно оно истекло. Например, CookiesIsExpired: 102 days, 1:31:07\. |
| EarlyRegistrationError | Ошибка вызвана задержкой в передаче запросов. Подождите несколько минут, и данные загрузятся. |
| WebsiteDoesNotExist | Такая площадка веб\-мастера не найдена. |
| WebsiteAdvCampaignDoesNotExist | Данный веб\-мастер не подключен к вашей программе. |

 

Если вы не можете локализовать ошибку или у вас возникли иные сложности, свяжитесь со специалистом отдела трекинга.
