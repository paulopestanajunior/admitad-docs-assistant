# How to add affiliate program description in Admitad

The description of your program is one of the key factors considered by publishers when deciding whether to join it or not.


Publishers can see program info in the following locations:

- Program card in the [Admitad Store](https://store.admitad.com/en/catalog/) catalog
- Program page

*In the version of the catalog for unauthorized users, the information about your program is updated once a day.*

## Adding program info


The way you should do it depends on how the program is managed.

- Your program is managed by a manager. In this case, ask your manager to add or edit your program's info.
- You manage the program on your own. In this case, you add program info by yourself.


To add or edit program info, go to Program information (Program → Program profile → Program information).


Below you'll find some tips for filling out all the fields in this section.


Having filled out all the fields, click Preview and check what a publisher will see.

## Name


Enter your company or brand name in the Name field. Your program will appear under this name in the [Admitad Store](https://store.admitad.com/en/catalog/) catalog.


Tips:

- Make sure the program name matches your brand's name. This will help publishers easily find your program using search.
- Don't add the project's brief description to the program name.
	- Wrong: *Sindbad — cheap flight tickets online*
	- Correct: *Sindbad*
- Don't add the domain name (e.g., *.com* etc.) to the program name.
	- Wrong: *Aliexpress.com*
	- Correct: *Aliexpress*


You can also indicate the [region](#geo-in-program-title) and [payment model](#postfixes-with-additional-information) in the program name. These indications will help publishers quickly decide whether your program is relevant to them.

- In the program name, you can indicate the region or regions where your program is available.


This indication will help publishers find out from which regions traffic is allowed.

| Program regions | Format | Example |
| --- | --- | --- |
| 1 region | 1 two\-letter region code | Romwe US |
| 2\-5 regions | Two\-letter region codes | Perfumes Club ES DE IT FR |
| 5\+ regions | Enter *Many GEOs*. | SHEIN Many GEOs |
| Worldwide | Enter *WW\** | Rentalcars WW |


\* You can use WW if your website or project is available in English alongside the local language. In this case, you should also add the program description and rules in English alongside the local language.


If you start several standalone programs for various regions, specify the two\-letter code of the corresponding region in the name of each program.


If the program only allows orders from one or several cities, indicate the country code in the name and specify the cities in the [program rules](https://support.admitad.com/hc/en-us/articles/4405920516113).

- Adding postfixes with additional information in the program name

Postfixes are used in programs of the categories *Online games*, *Mobile games*, *Financial programs*.


You can also provide additional details in the program info, namely:

	- Payment model (see abbreviations below)
	- Operating system your app supports (e.g., iOS or Android)
If you use a region code alongside a postfix, enter the program name in the format *Name \[postfix] {region code}*.

Allowed abbreviations:

| Acronym | Meaning |
| --- | --- |
| SOI | Single\-opt\-in |
| DOI | Double\-opt\-in |
| CPP | Cost per player |
| CPS | Cost per sale |
| CPC | Cost per click |
| CPL | Cost per lead |
| CPV | Cost per visit |
| CPI | Cost per Install |
| CPA | Cost per action.*Use this acronym for Cost per mailer, Cost per registration on app, and other models not listed here. Specify the exact model in the program description.* |
| API | Application programming interface |
| APK | Android package kit |

  
Other postfix requirements:

	- If several action types are subject to payment in the program, do the following:
	- - Specify all postfixes in brackets separating them by commas  
		*Example: Name \[CPC, CPL, CPS] RU*
		- Or only specify the main postfix  
		*Example: CPR and SOI — Name \[SOI] DE*
	- If only one of several actions is subject to payment in the program, specify postfixes separating them by slashes.  
	*Example: Name \[CPL/CPS] UA*
> *Example 1\. Banggood \[CPI, Android] DE*
> 
> 
> 
> 
> 	- *CPI (Cost Per Install) — only application installation is subject to payment*
> 	- *Android — Android app*
> 	- *DE — the program is only available in Germany*
> 
> *Example 2\. Star Stable \[SOI] DE AT CH*
> 
> 
> 
> 
> 	- *SOI. Registrations not requiring confirmation are subject to payment.*
> 	- *DE AT CH — the program is only available in Germany, Austria, and Switzerland.*

## Website URL


In the Website URL field, check that the link to your website is specified correctly.


The link must not contain any additional tags or parameters.


Publishers will use this link to visit your website and familiarize with products and services they'll promote.


Use a direct link without tags

## Possible spellings of a program name


In the Alternative program names field, specify possible spelling variants separated by commas.


Also add spellings in transliteration, those containing the most common typos, those typed in various keyboard layouts, etc.


This will help a publisher find your program in the Admitad Store even if they make a mistake in the brand name.

## Logo


Upload your company logo as an SVG file.

## Description


Information in the Description field is one of the driving factors that shape a publisher's decision when selecting a program to join.


It's important that your store and service appears with a catchy description, so you can spark a publisher's interest and encourage them to join your program.


The description must always be up\-to\-date: If you want to edit it, do it on your own or ask your manager.


If it is the first time you're adding the program information, you can use the description template suggested in the field.

Fill the description using the template:

- Brief description of your company should state your line of business, main products, and regions of operation. No more than 3\-4 lines.
- Negative keywords.  
Specify words and phrases forbidden for publishers' use in contextual advertising (e.g. brand name) if any.
- Benefits for consumers.  
Specify your company's advantages that differ you from the competitors (e.g., 24/7 free delivery or exclusive brands).
- Target audience portrait.  
Description of your typical customer: sex, age, occupation, marital status.
- Popular products.   
Specify best\-selling categories or products.
- Important information for publishers. Traffic sources with the highest CR, target audience, etc.
- Brand positioning recommendations.  
Advise publishers on what to emphasize, provide words and phrases required to use in the advertising.


If your program belongs in the "Online games", "Mobile programs", or "Financial programs" category, add the information specified below in the description.

- - Online games:
		- KPIs used to evaluate the traffic
		- test limits
		- a link to the ad creatives storage and rules for using them
	- Mobile programs:
		- KPIs used to evaluate the traffic
	- Financial programs
		- Scan copy of the bank license

## Brief description


Provide the brief program description of no more than 1\-2 lines in the Brief description field.


This description is used in newsletters to publishers in which we present new programs.


You can just copy 1\-2 sentences from the Description field.

### What to do after filling program description


After you've filled out all the fields:


1\. Click Preview to know how publishers will see the description on the page of your program.


2\. Click Save to save changes and finish editing.


Done! The description of your program will appear (be updated) on the program page and in the program card in [Admitad Store](https://store.admitad.com/en/catalog/).
