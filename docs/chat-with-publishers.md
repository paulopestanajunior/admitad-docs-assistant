# Chat with publishers

Use chat to reach out to publishers and resolve important questions faster. It will let you:

- Invite publishers to join your program
- Start a chat with publishers you are already cooperating with

## How to open the chat window


You can open a chat:

- [In the top bar of your account](https://support.mitgo.com/hc/en-us/articles/18501645585681#how-to-open-chat-in-the-account-header)
- [In the Report on publishers tab](https://support.mitgo.com/hc/en-us/articles/18501645585681#how-to-open-chat-in-statistics)
- [In the My publishers tab](https://support.mitgo.com/hc/en-us/articles/18501645585681#how-to-open-chat-in-the-publishers-section)
- [In the Publisher invitations tab](https://support.mitgo.com/hc/en-us/articles/18501645585681#how-to-open-chat-in-the-invitations-section)
- [In the Promo code requests tab](https://support.mitgo.com/hc/en-us/articles/18501645585681#how-to-open-chat-in-the-promo-code-requests-section)

### How to open the chat in the top bar of the account


In the upper right\-hand corner, click .


You will be automatically redirected to the page where you can start chats with your publishers. [Learn more](https://support.mitgo.com/hc/en-us/articles/18501645585681#how-to-start-chat-with-publishers)

### How to open the chat in Reports


1\. Go to the On publishers tab (Reports → On publishers).


2\. Select a publisher and click  next to their name 

  

The chat will open. That's where you can start talking. All previous messages will be saved in this chat.

### How to open the chat in Publishers


1\. Go to Connected (Publishers → Connected).


2\. Select a publisher and click next to their name 


The chat will open. That's where you can start talking. All earlier messages will be saved in this chat.

### How to open the chat in Publisher invitations


1\. Go to History of publisher invitations (Publishers → Publisher invitations → History of publisher invitations).


2\. Select a publisher and click  next to their name.


The chat will open where you will see the invitation and will be able to continue talking to the selected publisher.

### How to open the chat in Promo code requests


1\. Go to Promo code requests (Advertising tools → Promo code requests).


2\. Select a publisher and click  next to their name.


The chat will open where you'll be able to discuss the terms of the requested promo code with the publisher.

## How to start a chat with a publisher


If you don't have active chats yet, you will see the following after [the chat window opens](https://support.mitgo.com/hc/en-us/articles/18501645585681#how-to-open-chat):


You can only start a chat with a publisher you cooperate with.


To do that:


1\. In the search bar, start typing the publisher's username.


2\. Select the publisher in Your publishers on the left.


3\. The chat will open. That's where you can start talking.

## Possible restrictions

- [The Open chat button is inactive for some publishers](https://support.mitgo.com/hc/en-us/articles/18501645585681#open-chat-button-disabled)
- [The chat with the selected publisher doesn't open](https://support.mitgo.com/hc/en-us/articles/18501645585681#chat-with-publisher-does-not-open)

### The Open chat button is inactive for some publishers


This means these publishers have been blocked or deleted.


If you hover over the  button, you will see a hint:  
*Publisher deleted*

What to do: In chats with a blocked or deleted publisher, you can only view message history. You can't send a message to such a publisher.

### The chat with the selected publisher doesn't open


This means you don't cooperate with this publisher.


If you try to start a chat with a publisher from Your publishers, you will see the following error:  
*This advertiser doesn't cooperate with the publisher*

What to do: Invite the publisher to cooperate following [these instructions](https://support.mitgo.com/hc/en-us/articles/4417451509393).
