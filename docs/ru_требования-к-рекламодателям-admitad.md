# Требования к рекламодателям Admitad

*Перед началом работы рекомендуем ознакомиться со статьей [«Admitad для рекламодателей: основы»](https://support.admitad.ru/knowledge-base/article/admitad-%D0%B4%D0%BB%D1%8F-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BE%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D0%B5%D0%B9-%D0%BE%D1%81%D0%BD%D0%BE%D0%B2%D1%8B_1).*


Перед тем, как зарегистрироваться в Admitad, убедитесь, что ваша компания соответствует требованиям партнерской сети:

- у вашей компании есть собственный действующий сайт или мобильное приложение;
- тематика вашего сайта или приложения не запрещена законодательством вашей страны и правилами Admitad.


Остальные требования зависят от категории, к которой относится ваш сайт:

- [Интернет\-магазины](#ecom)
- [Онлайн\-сервисы](#services)
- [Путешествия, туризм](#travel)
- [Финансовые программы](#finance)
- [Онлайн\-игры](#games)
- [Мобильные программы](#mobile)

### Интернет\-магазины


Для запуска партнерской программы требуется:

- наличие на сайте корзины;
- возможность оплаты заказа на сайте;
- наличие доставки в пределах страны;


Не подходят для запуска программы:

- сайты с одним товаром;
- сложные b2b\-проекты с узкой целевой аудиторией.

### Онлайн\-сервисы


Для запуска партнерской программы требуется:

- Возможность покупки на сайте либо наличие формы регистрации с опцией валидации данных (например, данных банковской карты).


Не подходят для запуска программы:

- сложные b2b\-сервисы с узкой целевой аудиторией (кроме проектов с трафиком от 700 тыс. уникальных пользователей в месяц).

### Путешествия, туризм


Для запуска партнерской программы требуется:

- наличие на сайте формы\-заявки для получения услуги или корзины с возможностью оплаты.

### Финансовые программы


Для запуска партнерской программы требуется:

- наличие на сайте формы\-заявки для получения услуги;
- наличие лицензии.

### Онлайн\-игры


Нет специальных требований к функциональности проекта.

### Мобильные программы


Для запуска партнерской программы требуется:

- наличие приложения в Google Play или App Store;
- для отдельных сегментов — наличие платных функций внутри приложения.


Даже если ваша компания соответствует всем описанным выше требованиям, а ее тематика не запрещена в вашей стране, в некоторых случаях Admitad может отказать вам в сотрудничестве.  
Если у вас остались вопросы или нужна помощь с регистрацией, отправьте письмо на почту [support@admitad.com](mailto:support@admitad.com).
