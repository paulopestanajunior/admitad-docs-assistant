# Размещение рекламы на сторонних ресурсах

Вы можете размещать рекламу на платных внешних площадках, например, в Яндексе или VK Ads.


При запуске арбитражных кампаний на сторонних ресурсах указывайте корректные данные рекламодателя — иначе объявление может быть отклонено или заблокировано по закону о рекламе.

## Где найти данные рекламодателя


1\. Перейдите в раздел «Каталог товаров».


2\. Нажмите кнопку «Скачать товары» — откроется окно «Выгрузка товаров».


3\. В поле «Формат файла» выберите «YML».


4\. В поле «Магазин на маркетплейсе» выберите магазин, товары которого хотите продвигать.


5\. Скопируйте данные о рекламодателе, необходимые для запуска кампании на сторонних ресурсах и для соблюдения закона о рекламе.

## «Данные рекламодателя» в Яндекс Директе


В разделе с информацией о рекламодателе укажите:

- сведения о конечном рекламодателе — юридические данные продавца магазина;
- сведения о контрагенте (посреднике) конечного рекламодателя — юридические данные Такпродам;
- сведения о договоре с конечным рекламодателем — данные договора об организации распространения рекламы между продавцом магазина и Такпродам.
