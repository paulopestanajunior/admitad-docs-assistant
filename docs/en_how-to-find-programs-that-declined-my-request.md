# How to find programs that declined my request?

To find programs that declined to work with your ad space, go to the program [catalog](https://store.admitad.com/en/catalog/) and select Declined in the Status filter.


You'll see programs that declined your join request.


If an advertiser declined your ad space, you can no longer submit a join request to this program. But if you have several ad spaces, you can try to join this program with your other ad spaces.
