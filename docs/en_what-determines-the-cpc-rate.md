# What determines the CPC rate

The CPC (Cost Per Click) rate is the reward for a unique user click on an affiliate link.


The rate per click is dynamic and depends on several factors:

- Quality of traffic. The system analyzes user behavior after they go to the advertiser's website.
- User's country. Clicks from different countries can have different prices.
- Ad space topic. Relevant themed traffic usually has a higher CPC.
- User behavior. For example:
	- time spent on the advertiser's website,
	- interaction with content,
	- further user actions.
