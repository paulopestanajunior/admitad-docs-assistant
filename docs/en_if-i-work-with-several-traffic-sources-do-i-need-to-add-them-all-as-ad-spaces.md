# If I work with several traffic sources, do I need to add them all as ad spaces?

We recommend adding all traffic sources as ad spaces, especially if they belong to different types. However, in case you have several sites of the same type and with the same topic, for example, 10 Facebook groups about sports, you can add one ad space, and in the How will you promote the affiliate program? field explain that you are going to attract traffic from several similar ad spaces and insert links to them.

In any case, note that some programs do not accept traffic from unconnected ad spaces and have the right to reject all actions that were performed with the use of such traffic.
