# Доступы к Admitad API для рекламодателя

Всю детальную информацию по API и его использованию можно получить из [«Документации Admitad API»](https://developers.admitad.com/ru/knowledge-base/articles/for-advertisers).


Учетные данные, которые потребуются при работе с API, находятся на вкладке «API и приложения» в настройках аккаунта (значок аккаунта → Настройки профиля → API и приложения).


По умолчанию учетные данные скрыты в целях безопасности.  
Чтобы увидеть их, нажмите «Показать учетные данные».


Учетные данные API дают полный доступ к вашему аккаунту так же, как и ваш логин и пароль, поэтому никому их не передавайте.  
Если вы знаете или подозреваете, что доступ был украден, сбросьте данные. После этого будет сгенерирован новый client\_secret. Учтите, что при сбросе все приложения, которые используют это ключ, будут отключены.


Помимо учетных данных здесь представлен список приложений, которым вы разрешили доступ к своему аккаунту. Чтобы проверить, какие именно права доступа получило то или иное приложение, нажмите «Доступы».
