# Как сбросить двухфакторную аутентификацию для аккаунта Mitgo ID

Если вы утратили доступ к своему устройству аутентификации, вы можете сбросить двухфакторную аутентификацию, чтобы войти в свой аккаунт Mitgo ID.

Если в вашем аккаунте Mitgo ID было добавлено несколько устройств аутентификации и вы утратили доступ к ним всем, то, чтобы войти в аккаунт, вам нужно сбросить двухфакторную аутентификацию поочередно для каждого из этих устройств.

Чтобы сбросить двухфакторную аутентификацию:

- 1\. На форме для ввода кода из приложения Google Authenticator нажмите «Нет доступа к устройству».
- 1\. На форме для ввода кода из приложения Google Authenticator выберите устройство, для которого нужно сбросить двухфакторную аутентификацию, и нажмите «Нет доступа к выбранному устройству».

2\. Далее вам потребуются:

- Ключ для сброса, который вы сохранили при добавлении этого устройства аутентификации.  
*Без этого ключа невозможно самостоятельно сбросить двухфакторную аутентификацию. Если у вас его нет, то для восстановления доступа к аккаунту свяжитесь с поддержкой: [support@mitgo.com](mailto:support@mitgo.com)*.
- Код для сброса — нажмите «Запросить код»: письмо с кодом будет отправлено на ваш email.   
*Код из письма действует в течение 5 минут.*

3\. Заполните форму:

- введите код из полученного письма;
- введите ключ для сброса двухфакторной аутентификации.

Нажмите «Далее».

У вас есть 3 попытки, чтобы правильно ввести код и ключ для сброса. Если количество попыток превышено, вы сможете повторить попытку сброса двухфакторной аутентификации только на следующий день.

Готово, вы сбросили двухфакторную аутентификацию для выбранного устройства.

После этого:

- на ваш email придет письмо с подтверждением, что вы удалили устройство для двухфакторной аутентификации из аккаунта Mitgo ID;
- вы сможете войти в аккаунт Mitgo ID и перейти в нужный сервис из раздела [«Продукты Mitgo»](https://id.mitgo.com/services).
