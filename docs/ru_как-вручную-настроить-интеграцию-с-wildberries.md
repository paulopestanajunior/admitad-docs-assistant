# Как вручную настроить интеграцию с Wildberries

Интеграция с Wildberries (WB) позволяет продвигать ваши товары на Такпродам, а также:

- автоматически загружать список ваших товаров в каталог Такпродам и регулярно актуализировать его;
- получать информацию о заказах ваших товаров для ведения статистики и финансового учета.


В настоящее время доступна только ручная загрузка отчетов по внешнему трафику из вашего кабинета на Wildberries в систему Такпродам. Вы можете самостоятельно выгружать эти отчеты и отправлять их команде Такпродам или поручить эту задачу нашему менеджеру, предоставив ему ограниченный доступ к разделу «Продвижение» в кабинете на Wildberries. Такой доступ позволяет только просматривать и скачивать документы, без возможности редактировать.


Для ручной настройки интеграции:


1\. Авторизуйтесь в своем кабинете на [WB](https://seller-auth.wildberries.ru/?redirect_url=https%3A%2F%2Fcmp.wildberries.ru%2F). 


2\. В меню профиля нажмите «Интеграции по API».


3\. Нажмите «Создать токен».


4\. В открывшемся окне выполните следующие действия:


4\.1\. Нажмите «Для интеграции вручную» (1\).


4\.2\. Нажмите «Базовый токен» (2\).


4\.3\. Придумайте название токена для его идентификации (3\).


4\.4\. Выберите разделы «Контент», «Аналитика», «Цены и скидки», «Маркетплейс».   
К ним токен будет открывать доступ (4\).


4\.5\. Выберите опцию «Только чтение» для обеспечения безопасности. При этом доступе Такпродам сможет лишь просматривать и скачивать информацию, без возможности редактировать (5\).


4\.6\. Нажмите «Создать токен» (6\).


5\. Скопируйте созданный токен.


Важно! Токен можно скопировать только во время его создания. Сохраните его, так как повторное копирование невозможно: для безопасности данных токены хранятся в зашифрованном виде. Если вы не сохранили токен, удалите его и создайте новый.


6\. Авторизуйтесь на платформе [Такпродам](https://app.takprodam.ru/stores).


7\. Перейдите в раздел «Магазины» → «Маркетплейсы».


8\. Нажмите «Новая интеграция» — откроется страница «Настройка интеграции».


9\. Выберите маркетплейс WB и свой магазин на Такпродам для интеграции.


Добавить магазин на Такпродам можно в разделе «Магазины».


10\. В поле «API authorization token» вставьте токен, сгенерированный на WB.


11\. В поле «Ссылка на магазин» вставьте ссылку на ваш магазин на WB.


12\. Нажмите «Сохранить».


Готово! При успешной интеграции будет создана рекламная кампания. Вам останется только запустить ее — сделать это можно в разделе «Продвижение товаров» → «Рекламные кампании» на Такпродам.


Об автоматической настройке интеграции через расширение читайте в [статье](https://support.admitad.ru/knowledge-base/article/%D0%B0%D0%B2%D1%82%D0%BE%D0%BC%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F-%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0-%D0%B8%D0%BD%D1%82%D0%B5%D0%B3%D1%80%D0%B0%D1%86%D0%B8%D0%B8).
