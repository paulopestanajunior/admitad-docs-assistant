# Как перейти в настройки аккаунта

Чтобы перейти в настройки аккаунта, наведите курсор на поле с вашим логином и выберите «Настройки» в выпадающем списке.

  
  
Вы попадете на страницу  «Общие настройки аккаунта».

Переключайтесь между вкладками, чтобы попасть в нужный раздел настроек.
