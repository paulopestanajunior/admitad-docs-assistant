# Как сократить партнерскую ссылку?

О том, как получить партнерскую ссылку, читайте [здесь](https://support.admitad.ru/knowledge-base/article/%D0%B3%D0%B4%D0%B5-%D0%B2%D0%B7%D1%8F%D1%82%D1%8C-%D0%BF%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D1%83%D1%8E-%D1%81%D1%81%D1%8B%D0%BB%D0%BA%D1%83_1).


Для сокращения ссылок у нас есть инструмент [Shortlink](https://store.admitad.com/ru/webmaster/shortlink/) («Инструменты» — «Shortlink»). Просто вставьте туда ссылку и нажмите «Получить короткую ссылку».

Важно: инструмент сокращает только партнерские ссылки Admitad Partner Network.
