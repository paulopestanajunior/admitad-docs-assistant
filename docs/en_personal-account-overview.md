# Personal account overview

This account overview comes with a brief description of the sections and functionality of your Admitad account.

## Account header


The header is located in the upper part of the page and displayed on all account pages. It includes the following elements:

- [Account information](#account-info).
- [Buttons](#buttons) that allow you quick access to notifications, Inbox, Admitad support requests, and Help Center.
- [Menu](#menu) with account sections.

### Account information

- [Ad space](#ad-space)
- [Username](#login)
- [Account level](#account-level)

#### Ad space


The name of your [ad space](https://support.admitad.com/hc/en-us/articles/4403304880529#ad-space-en) is located at the center of the header.


To add a new ad space, click on the name of the current ad space, then click Add, then follow [these instructions](https://support.admitad.com/hc/en-us/articles/360019060997).


If you have several ad spaces, you can switch between them using the dropdown list:

#### Username


In the upper\-right corner you can find your Admitad username. Hover the cursor over the username to jump into [settings](#settings) or log out.

#### Account level


To the right of the username is the [account level](https://support.admitad.com/hc/en-us/articles/4403304880529#account-level-en), which is your rating in the Admitad system.

### Quick access buttons

#### Notifications

 This button can get you to the notifications from Admitad. If you have unread notifications, their number will be shown on the icon. Hover over the notification icon for a dropdown list of the five latest notifications. Learn more about this section in the [Publisher's Notification Center](https://support.admitad.com/hc/en-us/articles/360020412238).

#### Inbox

 This button can get you to the section with messages from advertisers or program managers. [Learn more about Inbox](https://support.admitad.com/hc/en-us/articles/7231920289297)

#### Support

 This button can get you to Admitad support requests. This is where you can send a request to support if you have a question or problem. [How to create and manage support requests](https://support.admitad.com/hc/en-us/articles/360020421978)

#### Help Center for publishers

 This button can get you to the [Help Center](https://support.admitad.com/hc/en-us) with system instructions and answers to frequently asked questions. The button isn't displayed on all account pages.

#### Programs


In this section, you can go to the Admitad Store catalog or suggest a new program for collaboration. The program menu is divided by tracking types:

Affiliate

- My affiliate programs 一 here you will find programs with link tracking that your ad space is connected to.
- All affiliate programs 一 a list of all partner programs with link tracking in the Admitad network.
- New affiliate programs 一 programs with link tracking that have appeared in the Admitad Store in recent months.
- Suggest a program 一 if you want to collaborate with an advertiser who does not yet have their own affiliate program, send a request and we will try to add them to the catalog.

Promo code:

- My promo code programs 一 programs with promo code tracking that your ad space is connected to.
- All promo code programs 一 a list of all programs with promo code tracking.
- New promo code programs 一 programs with promo code tracking that have appeared in the Admitad Store in recent months.

From partners:

- Affiliate programs from partners 一 here you will find programs with link tracking from our partners.
- Programs for Takefluence 一 programs for the Takefluence platform, maximally convenient for content creators on social media.


Learn more about Admitad Store in our [Overview of Admitad Store](https://support.admitad.com/hc/en-us/articles/360019182618).

#### Products

- Original. In this section, you can export a file with the products of any affiliate program. This function comes in handy if your ad space is an [affiliate store](https://support.admitad.com/hc/en-us/articles/4403304880529#affiliate-store-en) or price comparison service since it will facilitate the upload of program products to your website.
- Export templates. Here, you can create a template for a file to use in further product exports.


Read more about exporting original product feeds and templates in [How to export products](https://support.admitad.com/hc/en-us/articles/360019922878).

#### Reports


From this menu section, you can go to reports on ad spaces, programs, time, etc. The reports display the number of clicks and actions for your affiliate links, their total rewards, and other performance indicators. [Learn more about reports](https://support.admitad.com/hc/en-us/articles/360019210678)  
 

#### Payments


This section includes the following:

- Balance history. All transactions on balance changes (debits or credits). [Learn more about this section](https://support.admitad.com/hc/en-us/articles/4404837822225).
- Payment history. This section displays all your earlier withdrawals using [Admitad Earnings Wallet](https://support.admitad.com/hc/en-us/articles/7235247353617). Admitad Earnings Wallet is a payment service for Admitad partners that helps manage and conveniently withdraw earnings.
- Early payouts.\* This section leads to the Instant Payouts service.   
\*It is only displayed to those it is available for.

[How to withdraw funds from Admitad](https://support.admitad.com/hc/en-us/articles/9117474273809)  
 

## Settings


In Settings, you can update your details, edit or remove ad spaces, manage email notifications and payment details, create guest accounts, and set up APIs and two\-factor authentication.
