# How to set up two-factor authentication for your Mitgo ID account

Two\-factor authentication is an extra level of account protection that guarantees that only you can be granted access to your account, even if someone else has your password.  
  

## How two\-factor authentication works

After you enable two\-factor authentication for your account, you will need to enter the code from the Google Authenticator app:

- Upon every login to your account in any Mitgo product or service
- When making important changes to your account in order to verify that you are the person initiating them

Two\-factor authentication is set up for all Mitgo accounts in all products and services that you have. After you set up two\-factor authentication, you can use the Google Authenticator code for any of them.

 

## How to enable two\-factor authentication

1\. In the [Security](https://id.admitad.com/security) section, click the Two\-factor authentication field in Password and authentication.

2\. On the new page, click Enable and log in to your account again.

3\. Follow the instructions:

- Download the Google Authenticator app to your device.
	- [Link for iOS](https://apps.apple.com/en/app/google-authenticator/id388497605)
	- [Link for Android](https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2)
- Scan the QR on the screen. Your account will be added to the Google Authenticator app, and you will see a 6\-digit code that refreshes every 30 seconds.
- Save the key. Without it, you won't be able to reset two\-factor authentication or recover access to your account if you lose access to the authentication device.
- Enter this code in the Code field.
- Enter the name of the device on which the app is installed in the Device name field.

4\. Click Enable protection.

You’re all set! You’ve now enabled two\-factor authentication for your account.

You can add up to five devices for two\-factor authentication. To do so, click Add authentication device and complete [Steps 3 and 4 from the guide above](#add-device).

 

## How to disable two\-factor authentication

1\. In the [Security](https://id.admitad.com/security) section, click the Two\-factor authentication field in Password and authentication.

2\. On the new page, click Remove device.

3\. Enter the code from the Google Authenticator app to confirm the device’s removal.

If several devices have been added for two\-factor authentication, remove each of them to disable it.

You’re all set! You’ve now disabled two\-factor authentication for your account.  

 

## Problems with two\-factor authentication

- If you can't access the authentication device but have a reset key that you saved when adding this device, you can reset two\-factor authentication following [these instructions](https://support.mitgo.com/hc/en-us/articles/14252001778705).
- If you don't have a reset key and/or you have any problems with two\-factor authentication, send a message to [support@mitgo.com](mailto:support@mitgo.com).

In the message:

	- Describe the problem in detail
	- If requested by a support agent, provide information that could help identify you as the account owner.
