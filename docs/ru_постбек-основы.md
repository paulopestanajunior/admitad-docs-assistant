# Постбек: основы

Постбек — это эффективный инструмент для автоматизации передачи данных между системами.   
Если при работе с Такпродам вы используете собственную систему статистики действий, добавьте веб\-адрес своего сервера (постбек URL) в нашу систему и мгновенно получайте всю информацию о произошедших конверсиях. Данные передаются с помощью GET или POST HTTP\-запросов от сервера Такпродам на ваш сервер. Настройка постбека обеспечивает быстрый и точный сбор информации, что позволит вам улучшить анализ данных и увеличить конверсию рекламных объявлений.

## Как добавить постбек


1\. Войдите в меню профиля → «Постбек».


2\. Нажмите «Добавить постбек».


Вы можете добавить несколько постбек URL, настроив их под разные требования.


3\. На странице «Добавление постбека» заполните поля:

- Название постбека — укажите название для постбека, которое отражает суть передаваемых данных или содержит имя вашего сервера.
- HTTP метод — выберите удобный для вас метод получения информации.


Основное различие методов GET и POST заключается в способе передачи данных:  
\-метод GET отправляет вам всю собранную информацию как часть URL, добавляя параметры и их значения в хвост ссылки;  
\-метод POST передает данные в теле запроса, а не в самом URL.

- URL — укажите ссылку на ваш сервер, по которой будет приходить HTTP\-запрос.
- Параметр — выберите параметры отслеживания, которые будут передаваться на ваш сервер, и укажите для них названия, уже используемые в вашей системе.


4\. Нажмите «Тестировать», чтобы сразу проверить корректность работы постбека.

- Если тестирование прошло успешно, вы увидите уведомление об этом.
- Если есть ошибки в тестировании, проверьте, что введенный вами URL сервера правильный и доступный или обратитесь в поддержку на [support@takprodam.ru](mailto:support@takprodam.ru).


5\. Нажмите «Сохранить» в правом углу страницы — постбек появится в списке на странице «Постбек».


Управляйте постбеком на странице «Постбек».  
Чтобы включить или отключить постбек, используйте переключатель «Активация».  
Чтобы изменить настройки или удалить постбек, нажмите на три точки в строке постбека.

## Параметры запросов

- id — уникальный идентификатор конверсии.
- date — дата конверсии.
- subid — метка для детального отслеживания и анализа источников конверсии. Добавить subid можно при формировании партнерской ссылки по [API](https://api.takprodam.ru/doc/publisher/redocly/) или [вручную](https://support.admitad.ru/knowledge-base/article/subid-%D0%B2-%D0%BF%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%B8%D1%85-%D1%81%D1%81%D1%8B%D0%BB%D0%BA%D0%B0%D1%85-%D1%87%D1%82%D0%BE-%D1%8D%D1%82%D0%BE-%D0%B8-%D0%BA%D0%B0%D0%BA-%D0%B5%D0%B3%D0%BE-%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D1%8C_1).
- publisher\_fee — ваша комиссия.
- total\_cart — сумма корзины для CPA заказов.
- ip — ip адрес пользователя, который совершил клик.
- currency — валюта, в которой выполняется конверсия.
- status — статус конверсии. Возможны три значения:
	- pending — в ожидании,
	- approved — подтверждена,
	- declined — отклонена.

### Пример запроса


Вы выбрали параметры отслеживания, которые будут передаваться на ваш сервер: id, publisher\_fee, status.

HTTP\-запрос с использованием метода GET будет таким:   
https://example.com/foo/bar?conversion\_id\=564b70fd\-62f8\-4a31\-930a\-64f31a7a5c10\&amount\=10\.57\&status\=pending

HTTP\-запрос с использованием метода POST будет таким:

{


"conversion\_id":"564b70fd\-62f8\-4a31\-930a\-64f31a7a5c10",


"amount":10\.57,


"status":"pending"


}
