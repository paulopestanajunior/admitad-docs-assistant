# От чего зависит ставка CPC

Ставка CPC (Cost per Click) — это вознаграждение за уникальный клик пользователя по партнерской ссылке.


Ставка за клик динамическая и зависит от нескольких факторов:

- Качество трафика. Система анализирует поведение пользователей после перехода на сайт рекламодателя.
- Страна пользователя. Клики из разных стран могут иметь разную стоимость.
- Тематика площадки. Релевантный тематический трафик обычно имеет более высокий CPC.
- Поведение пользователей. Например:
	- время на сайте рекламодателя,
	- взаимодействие с контентом,
	- дальнейшие действия пользователей.
