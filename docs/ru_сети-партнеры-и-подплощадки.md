# Сети-партнеры и подплощадки

Если ваша площадка — это CPA\-сеть, то вы все равно можете сотрудничать с Admitad в качестве веб\-мастера. В этом случае ваш аккаунт получит метку «subnetwork» (т.е. сеть\-партнер), и вам нужно будет добавить к своей основной площадке (вашей CPA\-сети) все ресурсы собственных пользователей как подплощадки.

## Что такое подплощадка


Подплощадка — тип площадки, которая добавляется к вашей основной площадке, но фактически принадлежит пользователю вашей CPA\-сети.

Особенности подплощадок:

- Подплощадки добавляются к основной площадке только по API. [Как добавить подплощадки](#how-to-set-up)
- Все подплощадки автоматически подключаются ко всем программам, к которым подключена ваша основная площадка.
- Подплощадкам присваивается тип «Площадка сети\-партнера».
- Если рекламодатель отключит одну из подплощадок от программы за нарушения, ваша основная площадка, а также другие подплощадки все равно продолжат работать с программой.

## Зачем нужны подплощадки


Наличие подплощадок позволяет точно определить источник трафика в каждом конкретном случае. Благодаря этому, если владелец подплощадки привлекает некачественный трафик или нарушает правила, рекламодатель сможет определить это и отключить именно подплощадку нарушителя.


При этом отключение никак не повлияет на вашу основную площадку и те подплощадки, которые ничего не нарушали. Они продолжат работать с партнерской программой без изменений.

## Как настроить функционал


Перед настройкой попросите вашего менеджера Admitad добавить метку «subnetwork» к вашему аккаунту. Если у вас еще нет собственного менеджера, [обратитесь в поддержку Admitad](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D1%81%D0%B2%D1%8F%D0%B7%D0%B0%D1%82%D1%8C%D1%81%D1%8F-%D1%81-%D0%BF%D0%BE%D0%B4%D0%B4%D0%B5%D1%80%D0%B6%D0%BA%D0%BE%D0%B9-admitad_1) с этой же просьбой. 


Функционал настраивается по API: [документация по созданию подплощадок для сетей\-партнеров](https://developers.admitad.com/hc/ru/articles/7982118743441-Partner-networks).


Если у вас нет опыта работы c API Admitad, передайте [ссылку на API\-документацию](https://developers.admitad.com/hc/ru/articles/7982118743441) своему техническому специалисту.

## Как посмотреть список своих подплощадок


Для этого в личном кабинете перейдите в раздел «Площадки» (Настройки → Площадки).  
У ваших подплощадок будет указан тип «Площадка сети\-партнера».

  
Здесь же вы можете удалять подплощадки: для этого нажмите «Удалить» в колонке «Настройки».   
После удаления подплощадка будет отключена от всех программ и больше не будет отображаться в списке ваших площадок.

## Исключения


Некоторые рекламодатели не готовы работать с сетями\-партнерами. При попытке подключить подплощадки к такой программе по API запрос не будет обработан, и ни одна из подплощадок не будет подключена.


В этом случае вы можете работать с программой как обычный веб\-мастер, т.е. подключить к программе только основную площадку и получать партнерские ссылки для нее. [Как подключиться к партнерской программе](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%BF%D0%BE%D0%B4%D0%BA%D0%BB%D1%8E%D1%87%D0%B8%D1%82%D1%8C%D1%81%D1%8F-%D0%BA-%D0%BF%D0%B0%D1%80%D1%82%D0%BD%D0%B5%D1%80%D1%81%D0%BA%D0%BE%D0%B9-%D0%BF%D1%80%D0%BE%D0%B3%D1%80%D0%B0%D0%BC%D0%BC%D0%B5_1)
