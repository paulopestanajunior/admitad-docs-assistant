# Как передавать реферер клика в партнерской ссылке

Когда пользователь переходит по партнерской ссылке, которую вы разместили на своей площадке, информация о том, на какой странице он находился в этот момент, передается в заголовке запроса HTTP.

Эта информация называется «реферер клика» или  `referer`. Он может использоваться:

- Для диагностики проблем при отслеживании целевых действий.
- Для решения спорных ситуаций с рекламодателем.  
Когда вы работаете с рекламодателями, для них важно знать, откуда приходит ваш трафик. Это позволяет убедиться, что вы не нарушаете правила программы. Таким образом, вы можете предоставить значение  `referer` в качестве доказательства, чтобы разрешить возникшую спорную ситуацию с рекламодателем.

К сожалению, реферер клика не всегда фиксируется по умолчанию, поэтому рекомендуем вам самостоятельно встраивать его в ваши партнерские ссылки.

## Как добавить реферер клика в партнерскую ссылку

1\. Получите кодированное значение для URL\-ссылки на вашу площадку, с которой пользователь будет переходить по партнерской ссылке. 

Для этого можно использовать сервис по кодированию ссылок. Например, [URLEncoder](https://www.urlencoder.io/).

Пример:  
Ссылка на вашу площадку — *https://example.com*  
Кодированное значение — *https%3A%2F%2Fexample.com*

  
2\. Добавьте в партнерскую ссылку параметр  `ref` с кодированным значением URL\-ссылки на вашу площадку.

*Если партнерская ссылка содержит токен формата ?erid\=AbcdE1Fgh, то добавьте параметр `ref` после токена с помощью символа «\&».*

Пример добавления реферера клика в партнерскую ссылку без токена:

Партнерская ссылка — *https://ad.admitad.com/g/\<goto\_code\>*

Готовая ссылка с параметром  `ref` и кодированным значением URL\-ссылки —   
*https://ad.admitad.com/g/\<goto\_code\>/?ref\=https%3A%2F%2Fexample.com*

Пример добавления реферера клика в партнерскую ссылку с токеном:

Партнерская ссылка с токеном — *https://ad.admitad.com/g/\<goto\_code\>/?erid\=AbcdE1Fgh*

Готовая ссылка с параметром  `ref` и кодированным значением URL\-ссылки —   
*https://ad.admitad.com/g/\<goto\_code\>/?erid\=AbcdE1Fgh\&ref\=https%3A%2F%2Fexample.com*

Готово, вы добавили реферер клика в партнерскую ссылку.
