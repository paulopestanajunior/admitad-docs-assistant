# Как скачать информацию о товарах из каталога

1\. Перейдите в раздел «Каталог товаров».


2\. Нажмите «Еще фильтры» — откроется окно с настройками фильтра.


3\. Настройте параметры фильтра для поиска нужных товаров.


4\. Нажмите «Применить» — отобразится список товаров.


5\. Рядом с фильтром нажмите на иконку «Экспорт».


4\. В открывшемся окне выберите формат файла выгрузки: CSV, YML.


При выборе формата YML укажите, товары какого магазина на маркетплейсе вы хотите продвигать.


5\. Нажмите «Скачать».


Файл выгрузки формируется в течение 10 минут.  
YML\-файл автоматически сохраняется в папке «Загрузки» на вашем устройстве.   
CSV\-файл после формирования можно скачать в разделе «Выгрузки».
