# Сервисы сокращения ссылок для ВКонтакте

ВКонтакте режет редиректы для ссылок, укороченных с помощью некоторых сервисов (например, goo.gl), соответственно, действия по ним не отслеживаются.

Чтобы избежать такой ситуации, мы рекомендуем сокращать ссылки при помощи нашего инструмента Shortlink (Инструменты → [Shortlink](https://store.admitad.com/ru/webmaster/shortlink/)).
