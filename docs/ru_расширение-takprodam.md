# Расширение TakProdam

TakProdam setup integration helper — это расширение, с помощью которого вы можете:

- автоматически настраивать интеграцию вашего магазина на Такпродам с маркетплейсом,
- загружать список товаров в каталог для паблишеров.


Скачайте TakProdam setup integration helper из [магазина Chrome](https://chromewebstore.google.com/search/%D1%82%D0%B0%D0%BA%D0%BF%D1%80%D0%BE%D0%B4%D0%B0%D0%BC). Для работы с расширением используйте браузер Google Chrome.

## Интеграция с помощью расширения


Об автоматической настройке интеграции читайте [здесь](https://support.admitad.ru/knowledge-base/article/%D0%B0%D0%B2%D1%82%D0%BE%D0%BC%D0%B0%D1%82%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F-%D0%BD%D0%B0%D1%81%D1%82%D1%80%D0%BE%D0%B9%D0%BA%D0%B0-%D0%B8%D0%BD%D1%82%D0%B5%D0%B3%D1%80%D0%B0%D1%86%D0%B8%D0%B8).

## Добавление товаров с помощью расширения


Если у вашего магазина на Такпродам не настроена интеграция с маркетплейсом, используйте расширение, чтобы добавить товары для продвижения в каталог паблишера.


Актуально только для рекламной кампании «CPC с витриной» с маркетплейсом Avito.


Чтобы автоматически добавить товары, которые будут участвовать в рекламной кампании:


1\. Перейдите в раздел «Продвижение товаров» → «Рекламные кампании».


2\. Выберите нужную кампанию «CPC с витриной».


Если в списке нет такой кампании, настройте ее по [инструкции](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B7%D0%B0%D0%BF%D1%83%D1%81%D1%82%D0%B8%D1%82%D1%8C-%D1%80%D0%B5%D0%BA%D0%BB%D0%B0%D0%BC%D0%BD%D1%83%D1%8E-%D0%BA%D0%B0%D0%BC%D0%BF%D0%B0%D0%BD%D0%B8%D1%8E-%D1%81-%D0%BE%D0%BF%D0%BB%D0%B0%D1%82%D0%BE%D0%B9-%D0%B7%D0%B0-%D0%B2%D0%BE%D0%B2%D0%BB%D0%B5%D1%87%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9-%D0%BA%D0%BB%D0%B8%D0%BA).


3\. Кликните на строку кампании — вы перейдете на страницу «Настройка рекламной кампании».


4\. Откройте на Avito страницу товара, который нужно добавить для продвижения. Авторизация на маркетплейсе не требуется.


Если нужно добавить несколько товаров одного магазина — откройте на Avito страницу магазина со списком этих товаров.


4\. Включите расширение TakProdam. Иконка расширения находится справа от адресной строки — .


5\. В открывшемся окне расширения «Добавление товаров» проверьте, что информация о товаре указана верно.


Если вы хотите добавить не все товары со страницы магазина, исключите из списка те, которые не планируете продвигать.


6\. Нажмите «Продвигать» — данные товара загрузятся на странице «Настройка рекламной кампании» в блоке «Товары, которые будут участвовать в РК».


Если ссылка на товар не загрузилась автоматически, то добавьте ее вручную.


7\. В правом нижнем углу страницы нажмите «Сохранить изменения».


Готово! Вы добавили товары для кампании «CPC с витриной». При необходимости этот список можно редактировать: удалять неактуальные товары или добавлять новые.
