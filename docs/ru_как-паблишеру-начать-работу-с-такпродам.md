# Как паблишеру начать работу с Такпродам

Для работы на платформе Такпродам необходимо иметь статус юридического лица, ИП или быть самозанятым.

## Шаг 1\. Добавьте и подтвердите площадку

1. Откройте сайт и авторизуйтесь с помощью MitgoID. Если у вас еще нет MitgoID, зарегистрируйтесь.
2. В разделе «Площадки» нажмите «Добавить площадку».
3. Укажите информацию о площадке:
	- Название.
	- Тип площадки — платформа, где вы размещаете контент.
	- URL площадки.
	- Тип контента — основные темы контента, который вы размещаете на площадке. Рекомендуем выбирать не больше 3 тем — такие площадки вызывают больше доверия у продавцов. Если вы публикуете новости о скидках и акциях, выберите тип «Обзор покупок».
4. Подтвердите площадку по инструкции в интерфейсе: вставьте код в описание площадки. Если он не помещается в описание полностью, вставьте хотя бы первую часть. Вы сможете удалить этот код из описания, как только статус площадки изменится на Approved.
5. Нажмите «Добавить площадку».


Нажимая «Добавить площадку», вы соглашаетесь с условиями [оферты](https://storage.yandexcloud.net/uploads-prod/public/publisher_offer.pdf) Такпродам.


Готово — площадка добавлена в статусе Verification. В течение 48 часов мы проверим ее.

- В случае успешной проверки статус изменится на Approved. В этом статусе отредактировать информацию о площадке уже не получится. Если вы нашли ошибку и хотите ее исправить, пожалуйста, напишите нам в поддержку: support@takprodam.ru
- Если проверка покажет, что что\-то не так — статус изменится на Declined. Вы сможете отредактировать информацию о площадке и отправить ее на повторную проверку.

## Шаг 2\. Начните размещать партнерские ссылки

1. Перейдите в раздел «Каталог продуктов». Там вы увидите все доступные для продвижения товары.
2. Чтобы посмотреть карточку интересующего товара на маркетплейсе, нажмите на его название — справа откроется окно со ссылкой.
3. Выберите товар, который хотите прорекламировать у себя на площадке, и скопируйте партнерскую ссылку на него.
4. Разместите партнерскую ссылку у себя на площадке. Когда по вашей ссылке сделают заказ, у продавца будет от 1 до 4 недель на то, чтобы подтвердить продажу или отклонить ее, если товар вернули. Как только продавец подтвердит продажу, на ваш баланс будет начислена комиссия. Посмотреть баланс и вывести деньги вы можете в разделе «Финансы». [Подробнее](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B2%D1%8B%D0%B2%D0%B5%D1%81%D1%82%D0%B8-%D0%B4%D0%B5%D0%BD%D1%8C%D0%B3%D0%B8-%D0%B8%D0%B7-takprodam)


Если у вас есть возможность интегрироваться через API, рекомендуем выполнить следующий шаг.

## Шаг 3\. Настройте интеграцию с платформой Такпродам


Автоматизируйте процессы сбора и анализа данных с помощью API:

- динамически обновляйте партнерские ссылки на ваших ресурсах с учетом актуальной информации;
- автоматически получайте необходимые данные в реальном времени;
- расширяйте аналитические возможности;
- интегрируйтесь с другими системами.


Для доступа к [API](https://api.takprodam.ru/doc/publisher/redocly/) используйте API токен из личного профиля («Профиль» → «Общие настройки профиля» → «Генерация API токена»).
