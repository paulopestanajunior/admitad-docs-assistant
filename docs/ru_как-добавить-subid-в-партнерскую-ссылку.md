# Как добавить SubID в партнерскую ссылку

SubID – это параметр, который добавляется к партнерской ссылке в качестве метки для детального отслеживания и анализа источников трафика, эффективности кампании. Если вы используете SubID, то сможете увидеть статистику по нему в разделе «Статистика».

*Например, вы размещаете одну и ту же партнерскую ссылку на своей площадке несколько раз для разных постов. С помощью SubID можно определить, какой из постов более эффективный.*

## Особенности SubID:

- можно добавить к любой партнерской ссылке на товар в разделах «Каталог товаров» и «Акции и промокоды»;
- можно включить в полную ссылку или сгенерировать с ним короткую;
- подходит для партнерских ссылок всех рекламных кампаний;
- к ссылке можно добавить только один SubID;
- можно многократно создавать SubID для одной партнерской ссылки.

# Как добавить SubID в полную ссылку


1\. В списке товаров выберите товар для рекламы и кликните на него, чтобы перейти в его карточку.


2\. В поле «SubID» введите \&subid\=XXX, где XXX — это значение, которое вы хотите добавить к ссылке в качестве метки. Это может быть ID вашей площадки или рекламного объявления.


3\. Рядом с полем «Полная партнерская ссылка» нажмите на значок «Копировать» — в ссылку автоматически будет добавлен указанный параметр SubID.


4\. Разместите ссылку на товар на своей площадке.

*Пример полной ссылки с SubID:   
https://stage.takprdm.ru/0W4N0MO25v80/?redirectTo\=**https%3A%2F%2Fozon.ru%2Fproduct%2F1823976049\&erid\=123123123\&subid\=XXX*

# Как сгенерировать короткую ссылку с SubID


Короткую ссылку чаще используют для рекламы в соцсетях. Ее легче распространять, так как она помещается в текстовые поля с ограниченным количеством разрешенных символов.


1\. В списке товаров выберите товар для рекламы и кликните на него, чтобы перейти в его карточку.


2\. В поле «SubID» введите \&subid\=XXX, где XXX — это значение, которое вы хотите добавить к ссылке в качестве метки. Это может быть ID вашей площадки или рекламного объявления.


3\. Нажмите «Сгенерировать короткую ссылку» — появится поле «Короткая партнерская ссылка».


4\. Рядом с полем «Короткая партнерская ссылка» нажмите на значок «Копировать», чтобы скопировать сгенерированную короткую ссылку.


5\. Разместите ссылку на товар на своей площадке.


Для генерации короткой ссылки с SubID понадобится несколько секунд. На экране вы увидите оповещение, когда ссылка будет скопирована в буфер обмена.  
Если вы используете браузер Safari, дождитесь отображения короткой ссылки во всплывающем окне.
