# Mitgo ID: единый аккаунт в системе Mitgo

[Mitgo ID](https://id.admitad.com/personal-data) — это сервис для управления вашим единым аккаунтом в системе Mitgo.


В аккаунте Mitgo ID хранятся:

- ваши личные данные — имя, фамилия, дата рождения, страна резидентства;
- настройки безопасности;
- сведения о привязанных аккаунтах в соцсетях и сервисах;
- история сеансов и информация о них и другие данные.


Аккаунт в Mitgo ID позволяет получить доступ ко всем продуктам и сервисам Mitgo: Admitad, TakeAds, Earnings Wallet и другим.


Когда вы изменяете информацию или настройки в аккаунте Mitgo ID, эти изменения автоматически применяются ко всем аккаунтам в продуктах и сервисах Mitgo, которые вы используете.


В аккаунте Mitgo ID представлено несколько разделов:

- [Личные данные](https://id.admitad.com/personal-data) — здесь вы можете:
	- изменить свои личные данные — [инструкция](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%B8%D0%B7%D0%BC%D0%B5%D0%BD%D0%B8%D1%82%D1%8C-%D0%BB%D0%B8%D1%87%D0%BD%D1%8B%D0%B5-%D0%B4%D0%B0%D0%BD%D0%BD%D1%8B%D0%B5-%D0%B2-%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D0%B5-mitgo-id_1);
	- привязать свой аккаунт в соцсети или другом сервисе к аккаунту Mitgo ID — [инструкция](https://support.admitad.ru/knowledge-base/article/%D0%BA%D0%B0%D0%BA-%D0%BF%D1%80%D0%B8%D0%B2%D1%8F%D0%B7%D0%B0%D1%82%D1%8C-%D0%B8%D0%BB%D0%B8-%D0%BE%D1%82%D0%B2%D1%8F%D0%B7%D0%B0%D1%82%D1%8C-%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82-%D0%B2-%D1%81%D0%BE%D1%86%D1%81%D0%B5%D1%82%D0%B8-%D0%B8%D0%BB%D0%B8-%D1%81%D0%B5%D1%80%D0%B2%D0%B8%D1%81%D0%B5-%D0%BA-%D0%B0%D0%BA%D0%BA%D0%B0%D1%83%D0%BD%D1%82%D1%83-mitgo-id_1).
