# How to export coupons

The export function allows you to get structured coupon data for analysis, reporting, and campaign management. This guide provides a comprehensive overview of exporting coupon data from Takedeals.


To get coupon data:


1\. Go to the Deals \& Coupons section.


2\. Select the source you want to export coupons for.


Apply quick filters to narrow results by merchant, country, or coupon category, or use advanced filters for more specific criteria.


All selected filters will be applied to your exported data.


3\. In the upper right corner, click the export icon .


4\. Select the export format:

- CSV, XML, or JSON.
- Export via link: A special link to coupon data, updated approximately every 12 hours.


5\. Optional. Click Show SubID and add up to 3 SubIDs to be included in all tracking links in your export data.


6\. Submit data export:

- For CSV, XML, or JSON, click Export data. The file will download to your device.
- For Export via link, click Generate link.


Wait a few minutes for the link to be generated. If accessed too early, the data may still be processing. Copy the link once it is active.


All generated links are stored in the [Export link manager](https://app.takedeals.com/deals_coupons/manage) section, where you can manage them. The link to the page can be found in the top\-right corner on the Deals \& Coupons page.
