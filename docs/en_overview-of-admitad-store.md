# Overview of Admitad Store

Admitad Store is a catalog that contains a list of all the [affiliate programs](https://support.mitgo.com/hc/en-us/articles/4403304880529#affiliate-program-en) available in Admitad. There are three types of programs in Admitad Store distinguished by action tracking:

- [Affiliate programs](https://support.mitgo.com/hc/en-us/articles/4403304880529-%D0%93%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-Admitad-Partner-Network#affiliate-programs-en)
- [Promo code programs](https://support.mitgo.com/knowledge-base/article/admitad-partner-network-glossary_6#promo-code-programs-en)
- [Affiliate programs from partners](https://support.mitgo.com/hc/en-us/articles/4403304880529-%D0%93%D0%BB%D0%BE%D1%81%D1%81%D0%B0%D1%80%D0%B8%D0%B9-Admitad-Partner-Network#affiliate-programs-from-partners-en)


Specifics of the Admitad catalog:

- The programs are displayed as cards that contain the most important details about each program: its name, rating, key metrics, etc. [More on program cards](#card)
- On the left side of the page, there are filters you can use to search for a suitable program. [More on filters](#filters)
- In the top\-right corner, there are the sorting and Export buttons. [More on buttons](#sort-and-export)


If you're not logged in, the version of the catalog slightly differs from the one available to authorized users.

## Program card


The most important information about the program is displayed on its card. The program card consists of the following elements:

- [Program name and logo](#name-logo)
- [Badges](#badge)
- [Add to favorites icon](#favourite)
- [Rating](#rating)
- [Program metrics](#indicators)
- [More button](#more)

### Program name and logo


The name and logo are located in the middle of a card.


If you click on a program's name or logo, you will be redirected to the program page. There, you'll find its description, rules, and rates. You'll also be able to join the program on its page.


If you're already joined, you'll be able to find an [affiliate link](https://support.mitgo.com/hc/en-us/articles/4403304880529#affiliate-link-en) and [ad creatives](https://support.mitgo.com/hc/en-us/articles/360019765037) there.

  
 

### Badges


Badges are there to inform you about each program's special features right on the Admitad Store page.


For example, you can quickly check which programs offer mobile tracking if it matters for you that actions performed both on a website and through an app are paid for by the advertiser.


Hover a cursor over a badge to see a tooltip that explains what this badge indicates.

A program card might contain some of the following badges:

 — exclusive to Admitad Affiliate. This badge marks affiliate programs that are only available in Admitad Affiliate and not in any other affiliate network.

 — Admitad Affiliate recommends. This badge appears on the programs that our system selected specifically for you. The system analyzes the programs of publishers who are joined to the same programs as you. After that, the system picks the programs where publishers earned the most and recommends you those that you've not joined yet.

 — mobile app tracking. This badge marks the programs in which users' [actions](https://support.mitgo.com/hc/en-us/articles/4403304880529#action-en) are paid for regardless of whether they were performed through a website or a mobile app.  
Choose programs with this badge if most of your users use mobile devices for browsing. Then you can be sure that none of your actions get lost and your commission is paid in full.

 — increased rate. Programs in which a temporary rate increase is provided receive this label. To activate the increased rate, you must first accept the advertiser's offer in your account.

Payout frequency badges — reflect how often an advertiser pay publishers rewards for target actions made in their programs.


The program card may have one of the following badges:

 stands for regular payouts. This badge is assigned to programs where advertisers pay publishers for target actions on a regular basis.

 stands for delayed payouts. This badge is assigned to programs where advertisers may be late with payouts to publishers.

 stands for infrequent payouts. This badge is assigned to programs where advertisers are frequently late with payouts to publishers.


These programs' pages will have information about the frequency of payouts displayed as a hint to the right of the program's name.

### Add to favorites icon


You can click on the  icon to add programs to Favorites in order to have quick access to them whenever necessary.


Programs that you added to Favorites will be marked like this  in the Admitad Store catalog.


To see the programs from your Favorites list, use the Favorite filter located on the left side of the page.

### Program rating


Program ratings are based on the scores assigned to them by publishers that cooperate with those programs.


You can also evaluate a program if you have joined it. You can give a score either to a program in general or to its specific parameters: rates, payment terms, CR. [More on how to rate a program](https://support.mitgo.com/hc/en-us/articles/360019209398)  
 

### Program metrics


There are 4 metrics that are displayed in program cards:

- Commission — the reward you'll get for each [target action](https://support.mitgo.com/hc/en-us/articles/4403304880529#action-en) according to the [program rates](https://support.mitgo.com/hc/en-us/articles/4403304880529#rate-en).  
Commission can be stated either as a percentage of an order amount or as a fixed amount. The higher the rate the more you will earn if your user performs a target action and the advertiser confirms it. Click on the rate value to see what target actions the advertiser pays for and what the commission rates are.
- Payment — the average time it takes the advertiser to process, verify and pay for an action. [What does average hold time depend on?](https://support.mitgo.com/hc/en-us/articles/360019225718-After-the-user-performed-an-action-how-soon-will-I-get-paid-#action-processing-time)  
The shorter the payment time is the sooner you'll get paid for the confirmed actions.
- Conversion rate — the percentage of users that performed target actions after being redirected to the advertiser's website by publishers' affiliate links. [More on conversion rate (CR)](https://support.mitgo.com/hc/en-us/articles/4403304880529#cr-en)  
A low CR indicates that a lot of users click on the program's affiliate links but do not perform target actions, and it's not a very good sign. When the CR is high, it means that users are more willing to perform target actions.
- Confirmed — shows the percentage of actions confirmed by the advertiser out of all actions performed by users who followed the program's affiliate links. The value of this parameter is calculated based on the data from all the publishers that cooperate with the advertiser. [More on the approval rate](https://support.mitgo.com/hc/en-us/articles/4403304880529#approval-rate-en)  
A high confirmation rate indicates that the advertiser confirms a lot of actions. However, a low confirmation rate doesn't necessarily mean something is wrong with the program. A small percentage of conversions is perfectly normal for financial programs—it's one of the distinctive features of this segment.


To see linear charts reflecting the main metrics of the program, go to the Program Analytics section. To get there, click on a value of one of the parameters: Payment, Conversion rate, or Confirmed.


These metrics can be very useful when you are looking for a program to join: they help you to understand whether the program meets your criteria. It is, however, important to stay realistic and remember that no program is absolutely perfect. Therefore, when selecting affiliate programs to work with, you need to figure out your priorities: which parameters are crucial for you and which flaws you are willing to accept. [More on how to choose a program](https://support.mitgo.com/hc/en-us/articles/360019386758)


For instance, for those publishers that drive traffic through [arbitrage](https://support.mitgo.com/hc/en-us/articles/4403304880529#arbitrage-en), it is very important to get their payments ASAP to be able to invest more money into advertising. That is why they often prefer programs with the shortest payment time even if their commission rates or CR are slightly below average.  
 

### More button


The  button will redirect you to the program page, where you can read a detailed description of the program, learn more about its rules and rates, or obtain [affiliate links](https://support.mitgo.com/hc/en-us/articles/4403304880529#affiliate-link-en) and [ad creatives](https://support.mitgo.com/hc/en-us/articles/4403304880529#creative-en) if you have already joined it.  
 

### Join and Apply for moderation buttons

 if you see this button, it means that the program can be joined [automatically](https://support.mitgo.com/hc/en-us/articles/360019067057#auto-moderation). As long as your ad space complies with the rules, you can connect it to the program and get to work right away.

 this button means that ad spaces go through [manual moderation](https://support.mitgo.com/hc/en-us/articles/360019067057#manual-moderation) in the program. If you click on it, you will apply for joining the program.  
The advertiser will process your request within 5 days and either connect your ad space to their program or decline it. [More on how to join a program](https://support.mitgo.com/hc/en-us/articles/360019182978)  
 

### Get link button


If you have already joined the program, click  to generate an affiliate link.


In the Admitad Store catalog, you can generate either a [standard affiliate link](https://support.mitgo.com/hc/en-us/articles/4403304880529#affiliate-link-en) or a [deeplink](https://support.mitgo.com/hc/en-us/articles/360019283878).  
[More on all methods of generating affiliate links](https://support.mitgo.com/hc/en-us/articles/360019067357)  
 

## Catalog filters


Filters are located on the left side of the [Admitad Store](https://store.admitad.com/en/catalog/) page. Use them to search for programs that match certain parameters.


You can apply one or several filters at the same time.

- [Main Filter](#main-filter)
- [Instant Payout support](#instant-payout-filter)
- [Payment Frequency filter](#payment-frequency-filter)
- [Status filter](#status-filter)
- [Program categories filter](#program-categories-filter)
- [Traffic Sources filter](#traffic-sources-filter)
- [Tools filter](#tools-filter)

### Main Filter

- Search — enter a program's name here to find it in the Admitad Store catalog.
- Region — use this filter to find programs where advertisers are willing to pay for target actions from a specific country.  
To find programs in which only actions from a single country are paid for, select it in the filter and check the Local programs only option in the filter.  
All regions allowed in the program can be found in the Regions section. If a region is not specified on the program page, it means that the advertiser doesn't pay for target actions made by users from this region.
- Favorites — using this filter you can find programs that you have [added to Favorites](#favourite).
- Auto moderation — using this filter you can find programs without manual moderation. You can join these programs and start working with them right away.
- Manual moderation — using this filter allows you to find programs where the advertiser manually checks and accepts publishers' ad spaces. Moderation can take up to 5 business days.
- New program — using this filter you can find programs that were launched in Admitad recently.
- Exclusive program — using this filter you can find programs that are only available in Admitad and not in any other affiliate network.

### Filter "Instant Payout support"


Using this filter, you will find programs that support Instant Payout.


You can filter programs depending on the service version they support:

- Lite. Allows you to get your earnings before the advertiser pays for your actions. [Learn more about Instant Payout Lite](https://support.mitgo.com/hc/en-us/articles/4403557147665-Admitad-Instant-Payout#instant-payout-lite)
- Pro. Allows you to get your earnings before the advertiser pays for your actions. [Learn more about Instant Payout Pro](https://support.mitgo.com/hc/en-us/articles/4403557147665-Admitad-Instant-Payout#instant-payout-pro)
### Payment Frequency filter


This filter will help you find programs with the selected frequency of payments to publishers.

- Regular: If you choose this option, programs will be displayed in which advertisers regularly pay publishers for target actions.
- With delays: If you choose this option, programs will be displayed in which advertisers may sometimes be late with payments to publishers.
- Infrequent: If you choose this option, programs will be displayed in which advertisers often delay payouts to publishers.

### Status filter

- Joined — using this filter you can find programs your ad space is already connected to.
- Not joined yet — using this filter you can find programs your ad space is not yet connected to.
- Under moderation — using this filter you can find programs for which your join application is being reviewed by the advertiser.
- Declined — using this filter you can find programs for which the advertiser declined your join application.

### Program categories filter


Using this filter you can find programs that belong to a selected category.


In Admitad, each program belongs to a certain category which depends on the business area of the advertiser's company. E.g., the AliExpress program belongs to the Online Shops category, and Booking.com to Travel \& Tourism.


Use this filter to select affiliate programs that match the topic of your ad space the best, and to promote products that have higher chances of attracting your users' interest.  
 

### Traffic Sources filter


Using this filter you can find programs that allow a certain type of [traffic](https://support.mitgo.com/hc/en-us/articles/4403304880529#traffic-en).


For instance, if you are planning to promote advertiser's goods through email newsletters, tick the Email marketing option in the Traffic sources filter. You will see the list of programs that allow this type of traffic.


If your traffic comes from a source that is forbidden by the program rules, your actions will be declined, and your ad space will be disconnected from the program.

## Catalog operation buttons


There are two buttons in the top\-right corner:

- [Sorting](#sort)
- [Export](#export)

### Sorting


If you click , the drop\-down list will appear. It'll allow you to choose how programs in the Admitad Store catalog are sorted.


The default setting is sorting by rating (descending).

### Export


The Export button allows downloading the list of programs from the catalog as an XML\- or XLSX\-formatted file.


Click Export to select the file format and download.

  
In the new tab, the file preparation page will open.


Once the file is ready, you'll see a pop\-up window with the Download button.


If you applied filters before exporting, only programs that match the filtering conditions will be included in the export file.

- - id — the program identifier
	- name — the program name
	- site\_url — the program's website URL
	- status — an affiliate program activity status: active or not
	- campaign\_url — a link to the program page
	- date\_start — the date when the program launched in Admitad
	- gotolink — the standard [affiliate link](https://support.mitgo.com/hc/en-us/articles/4403304880529#affiliate-link-en) for your ad space leading to the program's main page
	- category — the category or subcategory of the program. If the program belongs to several categories, all of them will be listed
	- region — the region from which the traffic is allowed in the program. If there are several regions, all of them will be listed
	- action\_ranges — program [rates](https://support.mitgo.com/hc/en-us/articles/4403304880529#rate-en) in the "action name: commission" format  
	*Example:*Interior accessories NEW: 6\.90% \- 6\.90%
	- [CR](https://support.mitgo.com/hc/en-us/articles/4403304880529#cr-en) of the program
	- [eCPC](https://support.mitgo.com/hc/en-us/articles/4403304880529#e%D1%81pc-en) of the program
	- [EPC](https://support.mitgo.com/hc/en-us/articles/4403304880529#epc-en) of the program
	- avg\_hold\_time — [average time](https://support.mitgo.com/hc/en-us/articles/4403304880529#average-hold-time-en) in days it takes the advertiser to process actions
	- avg\_money\_transfer\_time — [average time](https://support.mitgo.com/hc/en-us/articles/4403304880529#average-payment-time-en) in days it takes the advertiser to pay for actions
	- rating — [program rating](#rating) in the catalog
	- negative\_keywords — [negative keywords](https://support.mitgo.com/hc/en-us/articles/4403304880529#negative-keywords-en) for contextual advertising. The list of negative keywords will be included in the file only if the advertiser has enabled the tool for monitoring contextual advertising BrandAuditor. If the list of negative keywords is missing from the file, you'll most likely find it on the program page or in the program rules.


The file will also contain information on:

- actions
- rates
- program's product feeds
